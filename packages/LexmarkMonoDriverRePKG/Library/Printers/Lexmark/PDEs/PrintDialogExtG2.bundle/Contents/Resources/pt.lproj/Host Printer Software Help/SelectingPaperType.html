Seleção de um tipo de papel

  1. No menu de opções, selecione Papel

  2. No menu Tipo de papel, selecione o tipo de papel que você deseja usar.

    Notas:
    • Antes de usar tipos de papel personalizados, configure-os na impressora. Para obter mais informações, consulte o Guia do usuário da impressora.
    • É possível usar as configurações da impressora.

  3. Clique em Imprimir.
