Bruke funksjonen Avansert layout

Med denne funksjonen kan du skrive ut dokumenter som hefter eller plakater.

  1. Velg Avansert layout på menyen Alternativer.

  2. Velg ett av følgende på menyen Layout:

    • Normal: Ingen alternativer for layout blir brukt for utskriftsjobben.

    • Hefte: Skriver ut dokumentet som et hefte.

    • Plakat: Skriver ut dokumentet som en plakat, ved å forstørre det ved å bruke flere sider.

  3. Hvis det er aktuelt, velger du hvordan dokumentet skal leses, på menyen Rekkefølge for dokumentlesing.

    • Venstre mot høyre

    • Høyre mot venstre

    Merk: Denne rekkefølgen avgjør også hvor innbindingskanten på heftet blir plassert. Hvis Venstre til høyre velges, plasseres innbindingskanten på venstre side av heftet.

  4. Klikk på Skriv ut.
