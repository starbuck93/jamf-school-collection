Välja papperstyp

  1. Välj Papper på alternativmenyn

  2. Välj den papperstyp som du vill använda på menyn Papperstyp.

    Anmärkningar:
    • Konfigurera de anpassade papperstyperna på skrivaren innan du använder dem. Mer information finns i skrivarens Användarhandbok.
    • Du kan använda skrivarinställningarna.

  3. Klicka på Skriv ut.
