Réglage des corrections de couleurs

Cette fonction ajuste les couleurs imprimées afin qu'elles correspondent à celles d'autres périphériques de sortie ou de systèmes de couleurs standard.

Remarque : Cette fonction est disponible uniquement pour les imprimantes couleur.

  1. Dans le menu des options, sélectionnez Couleur, puis cliquez sur Correction des couleurs.

  2. Dans le menu Correction des couleurs, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive la correction des couleurs.

    • Auto : applique une table de conversion des couleurs différente à chaque objet de la page imprimée.

    • Manuel : personnalise la table de conversion des couleurs appliquée à chaque objet de la page imprimée et active les paramètres de correction des couleurs RVB et CMJN.

    Remarque : La table de conversion des couleurs utilisée dépend du type de l'objet et de la manière dont est spécifiée la couleur de chaque objet.

  3. Si l'option Manuel est sélectionnée, choisissez une option pour chaque paramètre de correction des couleurs RVB et CMJN.

    Les options suivantes sont disponibles pour les corrections de couleurs RVB au niveau des images, du texte et des graphiques :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : n'applique aucune conversion des couleurs.

    • Affichage sRVB : permet d'imprimer les images pour qu'elles apparaissent telles qu'affichées sur un écran ordinateur. Ce paramètre est recommandé pour l'impression de graphiques.

    • Vives sRVB : permet d'imprimer les images pour qu'elles apparaissent plus saturées que sur l'écran d'un ordinateur. Ce paramètre est recommandé pour l'impression de graphiques et de texte à usage professionnel.

    • Affichage - Vrai noir : imprime les gris neutres à l'aide du toner noir.

    • Vives : imprime les images pour qu'elles apparaissent plus vives et plus saturées.

    Les options suivantes sont disponibles pour la correction des couleurs CMJN :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : n'applique aucune conversion des couleurs.

    • CMJN vives : imprime les images à l'aide d'une table de conversion des couleurs SWOP (Specifications for Web Offset Publications).

    • CMJN Europe : imprime les images à l'aide de la table de conversion des couleurs EuroScale.

    • CMJN US : imprime les images pour qu'elles apparaissent plus saturées à l'aide de la table de conversion des couleurs CMJN US.

  4. Cliquez sur Imprimer.
