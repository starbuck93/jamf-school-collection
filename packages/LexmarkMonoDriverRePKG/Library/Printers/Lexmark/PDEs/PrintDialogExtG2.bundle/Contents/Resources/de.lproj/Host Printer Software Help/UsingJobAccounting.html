Verwenden der Funktion "Auftragsabrechnung"

Mit dieser Funktion können Sie Druckaufträge nachverfolgen, indem Sie sie einem Benutzernamen oder einer Benutzer-ID, einem Kontocode oder einem Abteilungscode zuordnen Durch die Zuordnung spezifischer Kontoinformationen zu Druckaufträgen können Sie außerdem Druckauftragskontingente für Benutzer festlegen. Um Benutzerkontingente festzulegen, müssen Sie Device Quotas im Drucker installieren. Weitere Informationen finden Sie im Administrator-Handbuch für Device Quotas.

  1. Wählen Sie im Options-Menü die Option Auftragsabrechnung >Auftragsabrechnung aktivieren.

  2. Geben Sie Benutzernamen oder ID, Kontocode und Abteilungscode ein oder wählen Sie diese aus.

    Hinweis: Standardmäßig wird der Name des angemeldeten Benutzers verwendet. Um einen benutzerdefinierten Namen zu erstellen, deaktivieren Sie Angemeldeten Benutzer verwenden und geben Sie den Benutzernamen ein.

  3. Klicken Sie auf Drucken.
