カラー補正を設定する

この機能を使用すると、他の出力機器や標準カラーシステムに適合するように、ドキュメントの印刷に使用するカラーを調整できます。

メモ： この機能は、カラープリンタでのみ使用できます。

  1. ［オプション］メニューで、［カラー］を選択し、［カラー補正］をクリックします。

  2. ［カラー補正］メニューより、以下の中からいずれか 1 つを選択します。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • オフ - カラー補正を無効にします。

    • ［自動］ - この設定では、印刷されたページ上の各オブジェクトに対して異なる色変換テーブルが適用されます

    • ［手動］ - 印刷されるページ内の各オブジェクトに適用するカラー変換テーブルをカスタマイズし、RGB および CMYK カラー補正を設定できるようにします。

    メモ： オブジェクトの種類と各オブジェクトの色指定に応じて、異なるカラー変換テーブルを使用します。

  3. ［手動］を選択した場合、RGB および CMYK カラー補正を設定するためのオプションを選択します。

    次のオプションは、イメージ、文字、またはグラフィックスの RGB カラー補正で使用できます。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • ［オフ］ - カラー補正を適用しません。

    • ［sRGB 表示］ - コンピュータ画面の表示と同様の画像を印刷します。 この設定は、画像を印刷する場合に推奨されます。

    • ［sRGB 鮮明］ - コンピュータ画面の表示よりも高い彩度の画像を印刷します。 この設定は、業務用の画像やテキストを印刷する場合に推奨されます。

    • ［画面 - トゥルーブラック］ - ブラックトナーのみでグレーの中間色を印刷します。

    • ［鮮明］ - 明度と彩度がより高い画像を印刷します。

    次のオプションは CMYK カラー補正で使用できます。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • ［オフ］ - カラー補正を適用しません。

    • ［鮮明 CMYK］ - SWOP（Web Offset Publications）カラー補正テーブルの仕様を適用した画像を印刷します。

    • ［ユーロ CMYK］ - EuroScale カラー補正テーブルを使用した画像を印刷します。

    • ［US CMYK］ - US CMYK カラー変換テーブルを使用して、 より彩度が高い画像を印刷します。

  4. ［印刷］をクリックします。
