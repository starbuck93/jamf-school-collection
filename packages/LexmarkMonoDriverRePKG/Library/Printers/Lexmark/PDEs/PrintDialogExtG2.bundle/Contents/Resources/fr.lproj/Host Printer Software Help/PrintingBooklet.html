Impression d'un livret

Cette fonction permet d'imprimer un document de plusieurs pages qui peut être plié sous forme de livret.

  1. Dans le menu des options, sélectionnez Mise en page avancée.

  2. Dans le menu Mise en page, sélectionnez Livret.

  3. Dans le menu Feuilles par liasse, indiquez le nombre de pages incluses dans chaque liasse pliée.

    Remarque : Choisissez un nombre approprié pour cette option afin de faciliter le pliage de la liasse et de permettre l'alignement homogène des bords des feuilles.

  4. Si nécessaire, sélectionnez Conserver le format de page d'origine pour modifier le format du livret imprimé.

    Remarques :
    • Si l'option Conserver le format de page d'origine est sélectionnée, chaque page du livret présente le même format que le document d'origine. Par exemple, si le document d'origine fait 8,5 x 11 pouces, le livret est imprimé sur du papier 11 x 17 pouces. Le fait d'imprimer sur du papier 11 x 17 pouces permet de conserver les dimensions du document d'origine. Ce paramètre fonctionne mieux sur les documents d'origine avec un format de papier Statement, Lettre US, A5, A4 ou B5.
    • Si l'option Conserver le format de page d'origine n'est pas sélectionnée, les pages individuelles du livret sont mises à l'échelle afin d'être adaptées au format de papier du document d'origine.

  5. Si nécessaire, sélectionnez Inclure une couverture de livret pour activer l'ajout de pages de couverture, puis sélectionnez une ou plusieurs des options suivantes :

    • Imprimer la partie extérieure de la couverture avant

    • Imprimer la partie intérieure de la couverture avant

    • Imprimer la partie intérieure de la couverture arrière

    • Imprimer la partie extérieure de la couverture arrière

  6. Dans le menu Type de papier de la couverture, sélectionnez le type de papier de la couverture du livret.

  7. Dans le menu Source de papier de la couverture, sélectionnez le tiroir à utiliser pour la couverture du livret.

  8. Cliquez sur Imprimer.
