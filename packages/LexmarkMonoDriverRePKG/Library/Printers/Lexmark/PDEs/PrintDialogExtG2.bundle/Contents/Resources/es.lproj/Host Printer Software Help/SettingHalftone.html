Configuración de media tinta

Esta función permite seleccionar las pantallas de media tinta alternativas para la salida impresa.

  1. En el menú Opciones, seleccione Calidad.

  2. En el menú Media tinta, seleccione una de las siguientes opciones:

    • Usar valores de la impresora: Utilizar el valor especificado en la impresora.

    • Normal: Utilizar pantallas de media tinta estándar.

    • Detalle: Utilizar distintas pantallas de media tinta diseñadas para líneas de color finas y texto en lugar de uniformidad y suavidad.

  3. Haga clic en Imprimir.
