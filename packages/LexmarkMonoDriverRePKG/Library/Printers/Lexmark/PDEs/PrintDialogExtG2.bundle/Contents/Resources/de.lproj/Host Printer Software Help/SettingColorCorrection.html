Einstellen der Farbanpassung

Mit dieser Funktion können die gedruckten Farben an die Farben anderer Ausgabegeräte und Standardfarbsysteme angepasst werden.

Hinweis: Diese Funktion steht nur auf Farbdruckern zur Verfügung.

  1. Wählen Sie im Options-Menü die Option Farbeaus und klicken Sie dann auf Farbanpassung.

  2. Wählen Sie im Menü "Farbanpassung" eine der folgenden Optionen aus:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Aus: Deaktiviert die Farbanpassung.

    • Auto: Für jedes Objekt auf der gedruckten Seite wird eine andere Farbumwandlungstabelle verwendet.

    • Manuell: Mit dieser Option können die Farbumwandlungstabellen für die einzelnen Objekte der gedruckten Seite angepasst und die Einstellungen für die RGB- und CMYK-Farbanpassung aktiviert werden.

    Hinweis: Welche Farbumwandlungstabelle verwendet wird, hängt vom Objekttyp ab und davon, wie die jeweilige Farbe festgelegt wird.

  3. Wenn Manuell ausgewählt ist, wählen Sie für jede RGB- und CMYK-Farbanpassungseinstellung eine eigene Option aus.

    Zur RGB-Farbanpassung von Bildern, Texten und Grafiken sind folgende Optionen verfügbar.

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Aus: Es wird keine Farbanpassung verwendet.

    • sRGB-Anzeige: Die Bilder werden so gedruckt, wie sie auf dem Computerbildschirm angezeigt werden. Diese Einstellung ist besonders für Grafiken geeignet.

    • sRGB Leuchtend: Die ausgedruckten Bilder sind leuchtender als sie auf dem Computerbildschirm angezeigt werden. Diese Einstellung ist besonders für Unternehmensgrafiken und -texte geeignet.

    • Display - Echtes Schwarz: Der Drucker druckt neutrale Graustufen ausschließlich mit schwarzem Toner.

    • Leuchtend: Die gedruckten Bilder sind heller und gesättigter.

    Zur CMYK-Farbanpassung sind folgende Optionen verfügbar:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Aus: Es wird keine Farbanpassung verwendet.

    • Leuchtend CMYK: Die Bilder werden anhand der speziellen SWOP-Bildumwandlungstabelle (Specifications for Web Offset Publications) gedruckt.

    • Euro CMYK: Die Bilder werden anhand der Euroskala-Farbumwandlungstabelle gedruckt

    • US CMYK: Die gedruckten Bilder werden gesättigter, wenn die US- CMYK-Farbumwandlungstabelle verwendet wird.

  4. Klicken Sie auf Drucken.
