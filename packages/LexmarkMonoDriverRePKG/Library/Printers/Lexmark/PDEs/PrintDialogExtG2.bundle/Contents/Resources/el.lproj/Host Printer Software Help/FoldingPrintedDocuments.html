Δίπλωση εκτυπωμένων εγγράφων

Σημείωση: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη μονάδα δίπλωσης στον εκτυπωτή σας. Για περισσότερες πληροφορίες, ανατρέξτε στον Οδηγό χρήσης του εκτυπωτή.

  1. Στο μενού Επιλογές, επιλέξτε Τελείωμα.

  2. Στο μενού Δίπλωση, επιλέξτε τον τρόπο δίπλωσης ενός εγγράφου.

    • Απενεργοποίηση—Απενεργοποιεί τη λειτουργία δίπλωσης.

    • Δίπλωση—Διπλώνει αυτόματα έως πέντε φύλλα στη μέση, μικρή πλευρά προς τη μικρή πλευρά, για να σχηματιστεί μία δεσμίδα. Για εργασίες εκτύπωσης μεγαλύτερες των πέντε φύλλων, κάθε δεσμίδα των πέντε φύλλων είναι ένθετη στην επόμενη δεσμίδα.

    • Δίπλωση και συρραφή στο κέντρο—Διπλώνει αυτόματα το έγγραφο στη μέση, μικρή πλευρά προς τη μικρή πλευρά και, στη συνέχεια, συρράπτει κατά μήκος της δίπλωσης.

      Σημείωση: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη μονάδα συρραφής στον εκτυπωτή σας. Για περισσότερες πληροφορίες, ανατρέξτε στον Οδηγό χρήσης του εκτυπωτή.

    • Δίπλωση μεμονωμένων σελίδων—Διπλώνει αυτόματα μεμονωμένες σελίδες στη μέση, μικρή πλευρά προς τη μικρή πλευρά.

    • Τριπλή δίπλωση μεμονωμένων σελίδων—Διπλώνει αυτόματα μεμονωμένες σελίδες στα τρία, μικρή πλευρά προς τη μικρή πλευρά.

    Σημείωση: Αυτές οι επιλογές είναι διαθέσιμες μόνο σε ορισμένους εκτυπωτές.

  3. Κάντε κλικ στην επιλογή Εκτύπωση.
