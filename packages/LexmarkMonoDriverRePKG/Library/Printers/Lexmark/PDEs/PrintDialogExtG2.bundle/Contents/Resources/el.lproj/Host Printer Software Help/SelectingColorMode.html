Επιλογή λειτουργίας χρώματος

  1. Στο μενού Επιλογές, επιλέξτε Χρώμα.

  2. Για έγχρωμους εκτυπωτές, κάντε κλικ στην επιλογή Χρώμα.

  3. Στο μενού Λειτουργία χρώματος, επιλέξτε ένα από τα παρακάτω:

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Χρώμα—Έγχρωμη εκτύπωση του εγγράφου.

      Σημείωση: Αυτή η επιλογή είναι διαθέσιμη μόνο σε έγχρωμους εκτυπωτές.

    • Μονόχρωμη—Ασπρόμαυρη εκτύπωση του εγγράφου.

  4. Κάντε κλικ στην επιλογή Εκτύπωση.
