Impresión de pósteres

Esta función permite aumentar el tamaño de una salida impresa para varias veces el tamaño original. En función del tamaño de póster seleccionado, el documento se divide en un número igual de secciones, cada una de las cuales se imprime en una hoja de papel completa. Una vez que todas las secciones se imprimen, recorte los espacios en blanco adicionales alrededor de cada página y una las hojas individuales para crear un póster.

  1. En el menú Opciones, seleccione Presentación avanzada.

  2. En el menú Presentación, seleccione Póster.

  3. En el menú Tamaño, seleccione el tamaño del póster.

  4. En el menú Solapamiento entre páginas, seleccione la cantidad de solapamiento entre las páginas. Esta opción permite crear un espacio en el que puede adjuntar todas las páginas sin cubrir una parte de la imagen.

  5. Si es necesario, seleccione Imprimir marcas de recorte para imprimir una línea que sirve como guía para recortar el papel sobrante de cada sección del póster.

  6. Haga clic en Imprimir.
