Verwenden der Funktion "Erweitertes Layout"

Mit dieser Funktion können Sie Dokumente als Broschüre oder Poster drucken.

  1. Wählen Sie im Options-Menü die Option Erweitertes Layout.

  2. Wählen Sie im Menü "Layout" eine der folgenden Optionen aus:

    • Normal: Es werden keine Layoutoptionen auf den Druckauftrag angewendet.

    • Broschüre: Das Dokument wird als Broschüre gedruckt.

    • Poster: Das Dokument wird als Poster gedruckt, indem es auf mehrere Seiten vergrößert wird.

  3. Wählen Sie ggf. im Menü "Leserichtung des Dokuments" aus, wie das Dokument gelesen werden soll.

    • Links nach rechts

    • Rechts nach links

    Hinweis: Die Leserichtung bestimmt auch den Binderand der Broschüre. Wenn Links ausgewählt ist, befindet sich der Binderand auf der linken Seite der Broschüre.

  4. Klicken Sie auf Drucken.
