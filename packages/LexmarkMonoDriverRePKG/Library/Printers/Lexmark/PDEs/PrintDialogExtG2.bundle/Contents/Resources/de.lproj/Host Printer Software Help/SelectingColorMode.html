Auswählen des Farbmodus

  1. Wählen Sie im Options-Menü die Option Farbe.

  2. Klicken Sie bei Farbdruckern auf Farbe.

  3. Wählen Sie im Menü "Farbmodus" eine der folgenden Optionen:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Farbe: Das Dokument wird in Farbe gedruckt.

      Hinweis: Diese Option ist nur auf Farbdruckern verfügbar.

    • Schwarzweiß: Das Dokument wird in Schwarzweiß gedruckt.

  4. Klicken Sie auf Drucken.
