摺疊印出的文件

請注意： 請確認您的印表機已安裝摺疊分頁裝訂器。如需其他資訊，請參閱印表機用戶手冊。

  1. 在選項功能表中，選取輸出處理。

  2. 在「摺疊」功能表中，選取文件的摺疊方式。

    • 關閉—停用摺疊功能。

    • 摺疊—自動將紙張對摺，每次最多五張，短邊對短邊，製作成一冊。若列印工作的張數多於五張，每五張一冊的各冊將穿插在下一冊中。

    • 摺疊和中央裝訂—自動將文件對摺，短邊對短邊，然後沿著摺疊處裝訂。

      請注意： 請確認您的印表機已安裝分頁裝訂器。如需其他資訊，請參閱印表機用戶手冊。

    • 摺疊個別頁面—自動將個別的紙張對摺，短邊對短邊。

    • 將個別的頁面三摺—自動將個別的紙張三摺，短邊對短邊。

    請注意： 這些選項只適用於某些印表機。

  3. 按一下列印。
