Uso do recurso Layout avançado

Este recurso permite imprimir documentos como brochuras ou cartazes.

  1. No menu de opções, selecione Layout avançado.

  2. No menu Layout, selecione uma das opções a seguir:

    • Normal— Nenhuma opção de layout é aplicada ao trabalho de impressão.

    • Brochura— Imprime um documento como brochura.

    • Cartaz— Imprime um documento como cartaz, usando várias páginas para ampliá-lo.

  3. Se necessário, no menu Ordem de leitura do documento, selecione a forma como o documento será lido.

    • Da esquerda para a direita

    • Da direita para a esquerda

    Nota: Esta ordem também determina a borda de encadernação da brochura. Se a opção da esquerda para a direita for selecionada, a borda de encadernação será no lado esquerdo da brochura.

  4. Clique em Imprimir.
