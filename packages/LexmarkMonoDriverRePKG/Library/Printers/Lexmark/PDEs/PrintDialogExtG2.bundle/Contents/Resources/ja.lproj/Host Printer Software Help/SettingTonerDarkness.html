トナーの濃さを設定する

この機能を使用すると、印刷出力の濃淡を調できます。

  1. ［オプション］メニューで、［品質］を選択します。

  2. ［トナーの濃さ］メニューより、以下の中からいずれか 1 つを選択します。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • ［手動］ - 印刷出力のトナーの濃さをカスタマイズします。

  3. スライドバーを移動して、トナーの濃さの値を調整します。

  4. ［印刷］をクリックします。
