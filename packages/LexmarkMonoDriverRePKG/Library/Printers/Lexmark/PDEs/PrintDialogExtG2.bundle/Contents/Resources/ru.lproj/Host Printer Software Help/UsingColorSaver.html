Использование режима ColorSaver (экономия цветного тонера)

Уменьшение количества тонера, которое использует принтер при печати цветных изображений.

Примечание. Такая функция есть только в цветных принтерах.

  1. В меню параметров выберите Цвет, затем нажмите Цвет.

  2. В меню ColorSaverTM выберите одно из следующего:

    • Использовать настройки принтера — Использование настроек, заданных на принтере.

    • Вкл. — Включение ColorSaver.

    • Выкл. — Выключение ColorSaver.

    Примечание. Текст распечатывается с использованием количества тонера по умолчанию.

  3. Нажмите Печать.
