排紙トレイを選択する

メモ： プリンタ付属でない排紙トレイを使用するには、お使いのプリンタでオプションの排紙トレイがサポートされていることを確認します。 詳細はプリンタの『ユーザーズガイド』を参照してください。

  1. ［オプション］メニューで、［仕上げ］を選択します。

  2. 使用する排紙トレイを［排紙トレイ］メニューから選択します。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • ［標準排紙トレイ］ - プリンタの標準排紙トレイを使用します。

    • ［排紙トレイ [x]］ - お使いのプリンタで使用できるオプションの排紙トレイを使用します。

  3. ［印刷］をクリックします。
