Utilisation de la fonction « Imprimer et conserver »

Cette fonction vous permet de stocker de plusieurs façons vos travaux d'impression dans la mémoire de l'imprimante.

Remarque : Pour plus d'informations sur l'impression de travaux suspendus, reportez-vous au Guide de l'utilisateur de l'imprimante ou au Guide de l'administrateur de l'application que vous utilisez pour imprimer le travail.

  1. Dans le menu des options, sélectionnez Imprimer et suspendre.

  2. Sélectionnez la méthode d'impression du travail.

    • Impression confidentielle : permet de protéger par mot de passe les travaux d'impression confidentiels à l'aide d'un numéro d'identification personnel (PIN) à quatre chiffres.

    • Différer l'impression : stocke le travail d'impression dans la mémoire de l'imprimante pour l'imprimer ultérieurement.

    • Vérifier l'impression : imprime la première copie d'un travail d'impression comprenant plusieurs copies pour vérification. Les copies restantes sont mises en suspens jusqu'à ce que vous continuiez l'impression du travail ou que vous annuliez le travail.

    • Répéter l'impression : imprime immédiatement le travail d'impression et stocke une copie dans la mémoire de l'imprimante pour permettre l'impression ultérieure d'autres copies.

  3. Si nécessaire, modifiez le nom d'utilisateur associé au travail d'impression.

    Remarque : Le nom de l'utilisateur connecté est utilisé par défaut. Pour créer un nom personnalisé, décochez Utiliser le nom de l'utilisateur connecté, puis saisissez le nom d'utilisateur.

  4. Si l'option Impression confidentielle est sélectionnée, saisissez le code PIN associé au travail d'impression.

  5. Cliquez sur Imprimer.
