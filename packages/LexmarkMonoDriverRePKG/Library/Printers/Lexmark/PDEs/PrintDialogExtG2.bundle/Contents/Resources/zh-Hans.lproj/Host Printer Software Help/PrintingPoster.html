打印海报

此特性让您放大打印输出至原始尺寸的几倍。根据选择的海报尺寸，文档被分为相等的几个部分，每个部分打印在一整张纸上。当所有部分打印完成，修剪掉每个页面周围多余的白边，并将单独的几张纸拼在一起成为海报。

  1. 在选项菜单中，选择高级布局。

  2. 在“布局”菜单中，选择海报。

  3. 在“尺寸”菜单中，选择海报的尺寸。

  4. 在“页面之间重叠”菜单中，选择页面之间重叠的量。此设置让您创建空间，能够连接页面，而不会覆盖部分图像。

  5. 如果需要，请选择打印剪裁标记来打印线条作为从海报的每一部分修剪多余纸张的引导。

  6. 单击打印。
