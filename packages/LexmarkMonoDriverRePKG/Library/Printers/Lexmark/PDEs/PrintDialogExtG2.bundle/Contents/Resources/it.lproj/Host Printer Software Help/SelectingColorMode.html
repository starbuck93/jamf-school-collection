Selezione della modalità colore

  1. Nel menu Opzioni, selezionare Colore.

  2. Per le stampanti a colori, fare clic su Colore.

  3. Nel menu Modalità colore, selezionare una tra le seguenti opzioni:

    • Usa impostazione stampante: consente di utilizzare le impostazioni specificate nella stampante.

    • Colore: consente di stampare il documento a colori.

      Nota: Questa opzione è disponibile solo sulle stampanti a colori.

    • Monocromatica: consente di stampare il documento in bianco e nero.

  4. Fare clic su Stampa.
