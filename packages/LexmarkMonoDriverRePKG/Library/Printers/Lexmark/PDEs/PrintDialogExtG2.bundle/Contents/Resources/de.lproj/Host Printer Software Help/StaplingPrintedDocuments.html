Heften von Ausdrucken

Hinweis: Vergewissern Sie sich, dass Ihr Drucker über einen Hefter-Finisher verfügt. Weitere Informationen finden Sie im Benutzerhandbuch des Druckers.

  1. Wählen Sie im Options-Menü die Option Papierausgabe.

  2. Legen Sie im Menü "Heften" fest, wie das Dokument geheftet werden soll.

    Für Multipositionshefter und Locher-Finisher stehen folgende Optionen zur Verfügung:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Aus: Deaktiviert die Hefterfunktion.

    • Oben links: Eine Heftklammer oben links.

    • Oben rechts: Eine Heftklammer oben rechts.

    • Unten links: Eine Heftklammer unten links.

    • Unten rechts: Eine Heftklammer unten rechts.

    • Doppelt oben: Zwei Heftklammern oben.

    • Doppelt links: Zwei Heftklammern links.

    • Doppelt unten: Zwei Heftklammern unten.

    • Doppelt rechts: Zwei Heftklammern rechts.

    Für den Hefter/Locher-Finisher sind folgende Optionen verfügbar:

    Hinweis: Die Heftklammer wird an der linken Seitenkante positioniert.

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Aus: Deaktiviert die Hefterfunktion.

    • Auto: Die Heftung erfolgt automatisch.

    • Vorne: Eine Heftklammer an der Vorderseite.

    • Hinten: Eine Heftklammer an der Rückseite.

    • Doppelt: Zwei Heftklammern.

    Für den Hefter-Finisher sind folgende Optionen verfügbar:

    Hinweis: Die Heftklammer wird an der linken Seitenkante positioniert.

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Aus: Deaktiviert die Hefterfunktion.

    • Oben links: Eine Heftklammer oben links.

  3. Klicken Sie auf Drucken.
