Définition de l'amélioration de la résolution

Cette fonction permet d'améliorer les détails des images et du texte du travail d'impression en augmentant légèrement la densité des pixels en fonction du paramètre sélectionné.

Remarque : Cette option est disponible uniquement pour les imprimantes monochromes.

  1. Dans le menu des options, sélectionnez Qualité.

  2. Dans le menu Amélioration de la résolution, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive l'augmentation des pixels.

    • Polices : assombrit le texte du document imprimé. Ce paramètre affecte uniquement les données des polices.

    • Horizontalement : assombrit les lignes horizontales du texte et des images sur le document imprimé.

    • Verticalement : assombrit les lignes verticales du texte et des images sur le document imprimé.

    • Dans les deux sens : assombrit à la fois les lignes horizontales et verticales du texte et des images sur le document imprimé.

  3. Cliquez sur Imprimer.
