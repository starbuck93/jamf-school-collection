Setting halftone

This feature lets you select alternate halftone screens for the printed output.

  1. In the options menu, select Quality.

  2. In the Halftone menu, select one of the following:

    • Use Printer Setting—Use the setting that is specified in the printer.

    • Normal—Use standard halftone screens.

    • Detail—Use alternate halftone screens designed for thin color lines and text rather than uniformity and smoothness.

  3. Click Print.
