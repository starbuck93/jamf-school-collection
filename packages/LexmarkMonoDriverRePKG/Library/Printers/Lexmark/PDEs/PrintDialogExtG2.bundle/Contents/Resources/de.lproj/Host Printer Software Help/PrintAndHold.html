Verwenden der Funktion "Drucken und Zurückhalten"

Verwenden der Funktion "Drucken und Zurückhalten"

Mit dieser Funktion können Sie Druckaufträge auf unterschiedliche Weise im Druckerspeicher speichern.

Hinweis: Weitere Informationen zum Freigeben angehaltener Aufträge finden Sie im Benutzerhandbuch oder dem Administrator-Handbuch der Anwendung, die Sie zur Freigabe des Auftrags verwenden.

  1. Wählen Sie im Options-Menü die Option Drucken und Zurückhalten.

  2. Legen Sie fest, wie der Auftrag gedruckt werden soll.

    • Vertraulicher Druck: Schützt vertrauliche Druckaufträge durch eine vierstellige persönliche Identifikationsnummer (PIN).

    • Druck zurückbehalten: Speichert einen Druckauftrag im Druckerspeicher, um ihn zu einem späteren Zeitpunkt zu drucken.

    • Druck überprüfen: Druckt das erste Exemplar eines aus mehreren Exemplaren bestehenden Druckauftrags zur Überprüfung. Die restlichen Exemplare werden zurückgehalten, bis Sie sie zum Drucken freigeben oder den Druck abbrechen.

    • Druck wiederholen: Druckt den Auftrag direkt und speichert eine Kopie im Druckerspeicher, so dass zu einem späteren Zeitpunkt weitere Kopien gedruckt werden können.

  3. Bei Bedarf können Sie den Benutzernamen für den Druckauftrag ändern.

    Hinweis: Standardmäßig wird der Name des angemeldeten Benutzers verwendet. Um einen benutzerdefinierten Namen zu erstellen, deaktivieren Sie Angemeldeten Benutzer verwenden und geben Sie den Benutzernamen ein.

  4. Wenn Vertraulicher Druck ausgewählt ist, geben Sie die PIN für den Druckauftrag ein.

  5. Klicken Sie auf Drucken.
