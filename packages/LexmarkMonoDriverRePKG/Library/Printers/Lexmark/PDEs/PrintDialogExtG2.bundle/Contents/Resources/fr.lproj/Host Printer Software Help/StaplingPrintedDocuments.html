Agrafage de documents imprimés

Remarque : Assurez-vous qu'une unité de finition d'agrafage est installée sur votre imprimante. Pour plus d'informations, reportez-vous au Guide de l'utilisateur de l'imprimante.

  1. Dans le menu des options, sélectionnez Finition.

  2. Dans le menu Agrafage, sélectionnez la méthode d'agrafage souhaitée pour le document.

    Les options suivantes sont disponibles pour l'unité de finition d'agrafage/perforation multiposition :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive la fonction d'agrafage.

    • Haut gauche : place une agrafe dans l'angle supérieur gauche des pages.

    • Haut droit : place une agrafe dans l'angle supérieur droit des pages.

    • Bas gauche : place une agrafe dans l'angle inférieur gauche des pages.

    • Bas droit : place une agrafe dans l'angle inférieur droit des pages.

    • Double haut : place deux agrafes le long du bord supérieur des pages.

    • Double gauche : place deux agrafes le long du bord gauche des pages.

    • Double bas : place deux agrafes le long du bord inférieur des pages.

    • Double droit : place deux agrafes le long du bord droit des pages.

    Les options suivantes sont disponibles pour l'unité de finition d'agrafage/perforation :

    Remarque : L'agrafe est placée le long du bord gauche des pages.

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive la fonction d'agrafage.

    • Auto : place une agrafe automatiquement.

    • Avant : place une agrafe à l'avant.

    • Arrière : place une agrafe à l'arrière.

    • Double : place deux agrafes.

    Les options suivantes sont disponibles pour l'unité de finition d'agrafage :

    Remarque : L'agrafe est placée le long du bord gauche des pages.

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive la fonction d'agrafage.

    • Haut gauche : place une agrafe dans l'angle supérieur gauche des pages.

  3. Cliquez sur Imprimer.
