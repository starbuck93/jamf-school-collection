Sélection d'un mode couleur

  1. Dans le menu des options, sélectionnez Couleur.

  2. Pour les imprimantes couleur, cliquez sur Couleur.

  3. Dans le menu Mode couleur, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Couleur : imprime le document en couleur.

      Remarque : Cette option est disponible uniquement pour les imprimantes couleur.

    • Monochrome : imprime le document en noir et blanc.

  4. Cliquez sur Imprimer.
