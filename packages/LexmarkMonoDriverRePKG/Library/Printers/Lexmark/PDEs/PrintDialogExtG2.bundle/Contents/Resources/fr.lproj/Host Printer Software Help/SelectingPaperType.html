Sélection d'un type de papier

  1. Dans le menu des options, sélectionnez Papier

  2. Dans le menu Type de papier, sélectionnez le type de papier que vous souhaitez utiliser.

    Remarques :
    • Avant d'utiliser des types de papier personnalisés, configurez-les sur votre imprimante. Pour plus d'informations, reportez-vous au Guide de l'utilisateur de l'imprimante.
    • Vous pouvez utiliser les paramètres de l'imprimante.

  3. Cliquez sur Imprimer.
