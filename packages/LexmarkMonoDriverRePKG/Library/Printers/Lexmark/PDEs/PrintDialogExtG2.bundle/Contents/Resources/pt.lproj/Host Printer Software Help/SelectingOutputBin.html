Seleção de uma bandeja de saída

Nota: Para usar bandejas diferentes daquelas que foram fornecidas originalmente com a impressora, verifique se a impressora é compatível com bandejas opcionais. Para obter mais informações, consulte o Guia do usuário da impressora.

  1. No menu de opções, selecione Acabamento.

  2. No menu de Bandeja de saída, selecione a bandeja que deseja usar.

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Bandeja padrão- Usa a bandeja padrão da impressora.

    • Bandeja [x]-Usa uma bandeja opcional disponível para sua impressora.

  3. Clique em Imprimir.
