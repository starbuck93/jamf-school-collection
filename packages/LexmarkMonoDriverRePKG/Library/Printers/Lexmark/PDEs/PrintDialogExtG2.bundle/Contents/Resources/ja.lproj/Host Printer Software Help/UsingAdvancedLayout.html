［高度なレイアウト］機能を使用する

この機能を使用すると、小冊子またはポスターとしてドキュメントを印刷できます。

  1. ［オプション］メニューで、［高度なレイアウト］を選択します。

  2. ［レイアウト］メニューより、以下の中からいずれか 1 つを選択します。

    • ［標準］ - レイアウトオプションはどれも印刷ジョブに適用されません。

    • ［小冊子］ - ドキュメントを小冊子として印刷します。

    • ［ポスター］ - 複数のページを使用してドキュメントを拡大し、ポスターとして印刷します。

  3. 必要に応じて、［ドキュメントを読む順序］メニューでドキュメントを読む方法を選択します。

    • 左から右

    • 右から左

    メモ： また、この順序で小冊子の綴じる辺が決定されます。 ［左から右］を選択すと、綴じしろが小冊子の左側に来ます。

  4. ［印刷］をクリックします。
