设置色彩平衡

此特性让您通过单独增加或减少用于每个颜色平面的碳粉量来调整打印输出的颜色。

注意： 此特性仅在彩色打印机中可用。

  1. 在选项菜单中，选择颜色，然后单击色彩平衡。

  2. 在“色彩平衡”菜单中，选择下列选项之一：

    • 使用打印机设置—使用打印机中指定的设置。

    • 手动—定制青色、品红色、黄色和黑色碳粉在打印输出中的用量。

  3. 如果选择手动，请使用滑块来调整下列设置：

    • 红色—调整红色和青色之间的色彩平衡。

    • 绿色—调整绿色和品红色之间的色彩平衡。

    • 蓝色—调整蓝色和黄色之间的色彩平衡。

    • 白色—调整白色和黑色之间的色彩平衡。

  4. 单击打印。
