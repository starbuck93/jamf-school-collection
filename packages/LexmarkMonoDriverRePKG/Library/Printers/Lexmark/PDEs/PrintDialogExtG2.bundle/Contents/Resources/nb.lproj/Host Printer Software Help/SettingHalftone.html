Angi halvtone

Med denne funksjonen kan du velge alternative halvtoneskjermer for utskriftene.

  1. Velg Kvalitet på menyen Alternativer.

  2. Velg ett av følgende på menyen Halvtone:

    • Bruk skriverinnstilling: Bruker innstillingen som er angitt i skriveren.

    • Normal: Bruker standard halvtoneskjermer.

    • Detalj: Bruker alternative halvtoneskjermer utformet for tynne fargelinjer og tekst i stedet for ensartethet og jevnhet.

  3. Klikk på Skriv ut.
