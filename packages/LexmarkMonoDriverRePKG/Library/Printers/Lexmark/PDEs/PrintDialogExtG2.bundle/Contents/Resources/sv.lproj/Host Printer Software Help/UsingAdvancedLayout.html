Använda funktionen Avancerad layout

Med den här funktionen kan du skriver ut dokument som häften eller affischer.

  1. Välj Avancerad layout på alternativmenyn.

  2. Välj något av följande alternativ på menyn Layout:

    • Normal – Inga layoutalternativ används på utskriftsjobbet.

    • Häfte – Skriver ut ett dokument som ett häfte.

    • Affisch – Skriv ut ett dokument som en affisch genom att förstora det över flera sidor.

  3. Om det behövs väljer du hur dokumentet läses på menyn Dokumentläsningsordning.

    • Vänster till höger

    • Höger till vänster

    Obs! Den här ordningen bestämmer också häftets bindningssida. Om du väljer Vänster till höger hamnar bindningssidan till vänster på häftet.

  4. Klicka på Skriv ut.
