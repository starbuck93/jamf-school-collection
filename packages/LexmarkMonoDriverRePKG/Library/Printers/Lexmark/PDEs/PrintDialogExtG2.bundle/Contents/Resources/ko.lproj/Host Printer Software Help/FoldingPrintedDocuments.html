인쇄된 문서 접기

참고: 접기 마무리 장치가 프린터에 설치되어 있는지 확인하십시오. 자세한 내용은 프린터 사용 설명서를 참조하십시오.

  1. 옵션 메뉴에서 마무리를 선택합니다.

  2. 접기 메뉴에서 문서를 접는 방법을 선택합니다.

    • 꺼짐—접기 기능을 비활성화합니다.

    • 접기—최대 5장까지 짧은 가장자리에서 짧은 가장자리로 용지를 자동으로 반으로 접어 번들 하나를 만듭니다. 5장 이상을 인쇄하는 경우, 각 5장 번들이 다음 번들에 포개집니다.

    • 접기 및 가운데 스태플—짧은 가장자리에서 짧은 가장자리로 개별 용지를 자동으로 반으로 접습니다.

      참고: 스태플 마무리 장치가 프린터에 설치되어 있는지 확인하십시오. 자세한 내용은 프린터 사용 설명서를 참조하십시오.

    • 개별 페이지 접기—짧은 가장자리에서 짧은 가장자리로 개별 용지를 자동으로 반으로 접습니다.

    • 개별 페이지 3단 접기—짧은 가장자리에서 짧은 가장자리로 개별 용지를 자동으로 3등분으로 접습니다.

    참고: 이 옵션은 일부 프린터에서만 사용할 수 있습니다.

  3. 인쇄를 클릭합니다.
