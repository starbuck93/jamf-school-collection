Uso do recurso Contagem de trabalhos

Esse recurso permite rastrear trabalhos de impressão associando-as com um nome ou ID de usuário, um código de conta e um código de departamento. Ao associar trabalhos de impressão com informações de conta específicas, você pode aplicar cotas aos trabalhos de impressão que os usuários podem enviar. Para aplicar cotas aos usuários, instale Gestão de cotas na impressora. Para obter mais informações, consulte o Guia do administrador de Gestão de cotas.

  1. No menu de opções, selecione Contagem de trabalhos >Ativar a contagem de trabalhos.

  2. Digite ou selecione um nome ou ID de usuário, um código da conta e um código do departamento.

    Nota: Por padrão, o nome de usuário conectado é usado. Para criar um nome personalizado, desmarque Usar nome de usuário conectado e digite o nome de usuário.

  3. Clique em Imprimir.
