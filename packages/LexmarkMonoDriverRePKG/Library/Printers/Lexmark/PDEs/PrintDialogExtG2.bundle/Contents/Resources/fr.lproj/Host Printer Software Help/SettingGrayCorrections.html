Réglage de la correction des gris

Cette fonction permet de régler la luminosité et le contraste du document lors d'une impression en noir et blanc.

Remarque : Cette fonction est disponible uniquement pour les imprimantes monochromes.

  1. Dans le menu des options, sélectionnez Couleur.

  2. Dans le menu Correction des gris, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive la correction des gris.

    • Manuel : personnalise la luminosité et le contraste du travail d'impression.

  3. Si l'option Manuel est sélectionnée, utilisez les curseurs pour régler les paramètres suivants :

    • Luminosité : règle la luminosité du travail d'impression.

    • Contraste : règle le contraste du travail d'impression.

  4. Cliquez sur Imprimer.
