Impression d'une page de séparation

Utilisez cette fonction pour séparer un document d'un autre lors de l'impression de plusieurs documents ou copies du document dans un travail d'impression unique.

  1. Dans le menu des options, sélectionnez Finition.

  2. Dans le menu Alim. séparateur, sélectionnez le tiroir de la page de séparation.

  3. Dans le menu Page de séparation, sélectionnez l'endroit où imprimer une page de séparation.

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Aucune : n'imprime aucune page de séparation.

    • Entre les travaux : imprime une page de séparation une fois l'impression d'un travail terminée.

    • Entre les copies : imprime une page de séparation une fois l'impression d'une copie du travail d'impression terminée.

  4. Cliquez sur Imprimer.
