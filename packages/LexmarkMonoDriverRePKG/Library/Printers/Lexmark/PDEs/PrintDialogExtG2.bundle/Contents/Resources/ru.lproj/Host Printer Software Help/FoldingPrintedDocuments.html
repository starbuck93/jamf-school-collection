Фальцовка распечатанных документов

Примечание. В принтере должен быть установлен брошюровщик с функцией фальцовки. Для получения дополнительной информации см. Руководство пользователя принтера.

  1. В меню параметров выберите Послепечатная обработка.

  2. В меня "Фальцовка" выберите способ фальцовки документа.

    • Выкл. — Выключение функции фальцовки.

    • Фальцовка — Автоматическое формирование пачки, сгибанием пополам до пяти листов с совмещением коротких сторон. При печати заданий, включающих более 5 листов, каждая пачка из 5 листов вкладывается в следующую пачку.

    • Фальцовка и скрепка по центру — Автоматическое сгибание пополам с совмещением коротких сторон, и сшивание скрепками вдоль сгиба.

      Примечание. В принтере должен быть установлен брошюровщик. Для получения дополнительной информации см. Руководство пользователя принтера.

    • Фальцовка отдельных страниц — Автоматическое сгибание отдельных листов пополам с совмещением коротких сторон.

    • Тройной сгиб отдельных листов — Автоматическое сгибание отдельных листов втрое с совмещением коротких сторон.

    Примечание. Такие функции есть только в некоторых принтерах.

  3. Нажмите Печать.
