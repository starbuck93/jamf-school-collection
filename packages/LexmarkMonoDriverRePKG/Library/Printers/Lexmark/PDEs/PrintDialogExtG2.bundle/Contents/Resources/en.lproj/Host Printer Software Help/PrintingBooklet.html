Printing a booklet

This feature lets you print a multiple‑page document that can be folded into a booklet.

  1. In the options menu, select Advanced Layout.

  2. In the Layout menu, select Booklet.

  3. In the Sheets Per Bundle menu, specify the number of pages that are included in each folded bundle.

    Note: Setting this option to an appropriate number makes the bundle easy to fold and allows the paper edges to line up evenly.

  4. If necessary, select Maintain original page size to change the size of the printed booklet.

    Notes:
    • If Maintain original page size is selected, then each page of the booklet is the same size as the original document. For example, if the original document is 8.5 x 11 inches, then the booklet is printed on an 11 x 17‑inch paper. Printing on an 11 x 17‑inch paper retains the dimensions of the original document. This setting works best on original documents with statement, letter, A5, A4, or B5 paper size.
    • If Maintain original page size is not selected, then the individual pages of the booklet are scaled down to the paper size of the original document.

  5. If necessary, select Include booklet cover to enable adding of cover pages, and then select one or more of the following options:

    • Print on Outside of Front Cover

    • Print on Inside of Front Cover

    • Print on Outside of Back Cover

    • Print on Inside of Back Cover

  6. In the Cover Paper Type menu, select the paper type of the booklet cover.

  7. In the Cover Paper Source menu, select the tray to use for the booklet cover.

  8. Click Print.
