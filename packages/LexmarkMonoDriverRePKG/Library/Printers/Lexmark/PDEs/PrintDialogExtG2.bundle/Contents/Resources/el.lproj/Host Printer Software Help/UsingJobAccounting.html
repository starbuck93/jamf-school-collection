Χρήση της λειτουργίας Λογιστική καταγραφή εργασιών

Η λειτουργία αυτή σάς επιτρέπει να παρακολουθείτε εργασίες εκτύπωσης, με τη συσχέτισή τους με ένα όνομα ή αναγνωριστικό χρήστη, κωδικό λογαριασμού και κωδικό τμήματος. Συσχετίζοντας τις εργασίες εκτύπωσης με συγκεκριμένες πληροφορίες λογαριασμού, μπορείτε να επιβάλετε όρια στις εργασίες εκτύπωσης που μπορούν να αποστείλουν οι χρήστες. Για να επιβάλετε όρια στους χρήστες, εγκαταστήστε τα Όρια συσκευής στον εκτυπωτή. Για περισσότερες πληροφορίες, ανατρέξτε στον Οδηγό διαχειριστή ορίων συσκευής.

  1. Στο μενού Επιλογές, επιλέξτε Λογιστική καταγραφή εργασιών >Ενεργοποίηση λογιστικής καταγραφής εργασιών.

  2. Πληκτρολογήστε ή επιλέξτε ένα όνομα ή αναγνωριστικό χρήστη, κωδικό λογαριασμού και κωδικό τμήματος.

    Σημείωση: Από προεπιλογή, χρησιμοποιείται το όνομα χρήστη με το οποίο έχει γίνει σύνδεση. Για να δημιουργήσετε ένα προσαρμοσμένο όνομα, διαγράψτε τα περιεχόμενα του πεδίου Χρήση ονόματος συνδεδεμένου χρήστη και, στη συνέχεια, πληκτρολογήστε το όνομα χρήστη.

  3. Κάντε κλικ στην επιλογή Εκτύπωση.
