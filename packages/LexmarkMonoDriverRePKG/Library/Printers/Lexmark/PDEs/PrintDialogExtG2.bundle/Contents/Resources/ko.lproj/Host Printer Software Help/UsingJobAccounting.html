작업 계정 기능 사용

이 기능을 사용하면 인쇄 작업을 사용자 이름이나 ID, 계정 코드 및 부서 코드와 연결하여 추적할 수 있습니다. 인쇄 작업을 특정 계정 정보와 연결하여 사용자가 보낼 수 있는 인쇄 작업에 할당량을 적용할 수 있습니다. 할당량을 사용자에게 적용하려면 프린터에 장치 할당량을 설치합니다. 자세한 내용은 장치 할당량 관리자 설명서를 참조하십시오.

  1. 옵션 메뉴에서 작업 계정 >작업 계정 활성화를 선택합니다.

  2. 사용자 이름이나 ID, 계정 코드 및 부서 코드를 입력하거나 선택합니다.

    참고: 기본적으로 로그인한 사용자 이름이 사용됩니다. 사용자 정의 이름을 만들려면 로그인 사용자 이름 사용을 지운 후 사용자 이름을 입력합니다.

  3. 인쇄를 클릭합니다.
