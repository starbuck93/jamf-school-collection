Utilisation de la fonction de mise en page avancée

Cette fonction permet d'imprimer des documents sous forme de livrets ou d'affiches.

  1. Dans le menu des options, sélectionnez Mise en page avancée.

  2. Dans le menu Mise en page, sélectionnez l'une des options suivantes :

    • Normal : aucune option de mise en page n'est appliquée au travail d'impression.

    • Livret : imprime un document sous forme de livret.

    • Affiche : imprime un document sous la forme d'une affiche en l'agrandissant sur plusieurs pages.

  3. Si nécessaire, dans le menu Ordre de lecture des documents, sélectionnez la manière dont le document est lu.

    • Gauche à droite

    • Droite à gauche

    Remarque : Cet ordre détermine également le bord de reliure du livret. Si l'option Gauche à droite est sélectionnée, alors le bord de reliure est situé sur le côté gauche du livret.

  4. Cliquez sur Imprimer.
