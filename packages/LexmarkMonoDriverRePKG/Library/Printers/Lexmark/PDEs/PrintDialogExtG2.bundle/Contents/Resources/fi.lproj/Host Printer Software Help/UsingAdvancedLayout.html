Asettelun lisäasetukset -toiminnon käyttäminen

Tämän toiminnon avulla voit tulostaa asiakirjat vihkoina tai julisteina.

  1. Valitse asetusvalikossa Asettelun lisäasetukset.

  2. Valitse Asettelu-valikossa jokin seuraavista:

    • Normaali – tulostustyössä ei käytetä asetteluasetuksia.

    • Vihko – tulosta asiakirja vihkona.

    • Juliste – tulosta asiakirja julisteena suurentamalla se usealle sivulle.

  3. Valitse tarvittaessa asiakirjan luentajärjestys Asiakirjojen luentajärjestys -valikossa.

    • Vasemmalta oikealle

    • Oikealta vasemmalle

    Huomautus: Tämä järjestys määrää myös vihkon sidontareunan. Jos Vasemmalta oikealle on valittuna, sidontareuna on vihkon vasemmalla puolella.

  4. Valitse Tulosta.
