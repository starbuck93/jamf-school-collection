Настройка цветокоррекции

Функция выполняет регулировку цветопередачи при печати в соответствии с цветами, воспроизводимыми на других устройствах вывода, или цветами стандартных цветовых систем.

Примечание. Такая функция есть только в цветных принтерах.

  1. В меню параметров выберите Цвет, затем нажмите Цветокоррекция.

  2. В меню "Цветокоррекция" выберите одно из следующего:

    • Использовать настройки принтера — Использование настроек, заданных на принтере.

    • Выкл. — Отключение функции цветокоррекции.

    • Авто — К каждому объекту на распечатываемой странице применяется отдельная таблица преобразования цветов.

    • Вручную — Пользовательская настройка таблиц преобразования цветов, применяемых к каждому объекту на печатной странице, включение цветокоррекции RGB и CMYK.

    Примечание. Таблица преобразования цветов используется в зависимости от типа объекта и способа задания цвета для этого объекта.

  3. При выборе Вручную следует также выбрать значение для каждого параметра цветокоррекции RGB и CMYK.

    Для цветокоррекции RGB изображений, текстов и графики доступны следующие значения:

    • Использовать настройки принтера — Использование настроек, заданных на принтере.

    • Выкл. — Цветокоррекция не применяется.

    • Дисплей sRGB — Печать изображений, как они выглядят на мониторе компьютера. Такая настройка рекомендуется при печати графики.

    • Яркие цвета sRGB — Печать изображений в более насыщенных цветах, чем они выглядят на мониторе компьютера. Такая настройка рекомендуется при печати деловой графики и текста.

    • Отображение ‑ Истинный черный — Печать нейтральных серых оттенков с использованием только черного тонера.

    • Яркие цвета — Печать изображений в более ярких и насыщенных цветах.

    Для цветокоррекции CMYK доступны следующие значения:

    • Использовать настройки принтера — Использование настроек, заданных на принтере.

    • Выкл. — Цветокоррекция не применяется.

    • Яркие цвета CMYK — Печать изображений с использованием спецификаций для таблицы преобразования цветов SWOP.

    • Euro CMYK — Печать изображений с использованием таблицы преобразования цветов Euroscale.

    • US CMYK — Печать изображений в более насыщенных цветах при использовании таблицы преобразования цветов U.S. CMYK.

  4. Нажмите Печать.
