Uso do recurso "Imprimir e reter"

Esse recurso permite que você armazene trabalhos de impressão na memória da impressora de várias maneiras.

Nota: Para obter mais informações sobre a liberação de trabalhos de impressão retidos, consulte o Guia do usuário da impressora ou o Guia do administrador do aplicativo que você usa para liberar o trabalho.

  1. No menu de opções, selecione Imprimir e reter.

  2. Selecione como imprimir o trabalho.

    • Impressão confidencial-Protege por senha trabalhos de impressão confidenciais com um número de identificação pessoal de quatro dígitos (PIN).

    • Manter impressão-Armazena um trabalho de impressão na memória da impressora para imprimi-lo mais tarde.

    • Verificar impressão-Imprime a primeira cópia de um trabalho de impressão com várias cópias para verificação. As cópias restantes são retidas até serem liberadas para impressão ou canceladas.

    • Repetir impressão-Imprime o trabalho imediatamente e armazena uma cópia na memória da impressora para o caso de mais cópias precisarem ser impressas posteriormente.

  3. Se necessário, altere o nome de usuário associado ao trabalho de impressão.

    Nota: Por padrão, o nome de usuário conectado é usado. Para criar um nome personalizado, desmarque Usar nome de usuário conectado e digite o nome de usuário.

  4. Se a opção Impressão confidencial for selecionada, insira o PIN associado ao trabalho de impressão.

  5. Clique em Imprimir.
