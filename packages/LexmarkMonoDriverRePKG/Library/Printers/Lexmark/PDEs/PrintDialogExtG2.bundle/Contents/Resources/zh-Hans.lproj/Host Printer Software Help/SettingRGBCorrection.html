设置 RGB 修正

此特性让您调整打印输出的明亮度、对比度和饱和度。

注意： 此特性仅在彩色打印机中可用。

  1. 在选项菜单中，选择颜色，然后单击 RGB 修正。

  2. 在“RGB 修正”菜单中，选择下列选项之一：

    • 使用打印机设置—使用打印机中指定的设置。

    • 手动—定制打印机输出的明亮度、对比度和饱和度。

  3. 如果选择手动，请使用滑块来调整下列设置：

    • 明亮度—调整打印输出的明亮度。

    • 对比度—调整打印输出的对比度。

    • 饱和度—调整打印输出的逼真度和颜色浓度。

  4. 单击打印。
