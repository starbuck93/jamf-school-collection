Impressão de uma página separadora

Use esse recurso para separar um documento de outro ao imprimir vários documentos ou cópias do documento em um único trabalho de impressão.

  1. No menu de opções, selecione Acabamento.

  2. No menu de Origem do separador, selecione a bandeja da página separadora.

  3. No menu de Página separadora, selecione quando deseja imprimir uma página separadora.

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Nenhum- não imprime uma página separadora.

    • Entre trabalhos- Imprime uma página separadora depois que um trabalho de impressão terminar a impressão.

    • Entre cópias- Imprime uma página separadora depois que uma cópia do trabalho de impressão terminar de imprimir.

  4. Clique em Imprimir.
