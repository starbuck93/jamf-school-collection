Использование функции "Расширенный макет"

Эта функция позволяет распечатывать документы в виде буклетов или постеров.

  1. В меню параметров выберите Расширенный макет.

  2. В меню "Макет" выберите одно из следующего:

    • Обычная — Для задания на печать не применяются никакие параметры компоновки.

    • Брошюра — Печать документа в виде брошюры.

    • Плакат — Печать документа в виде плаката путем его увеличения с использованием нескольких страниц.

  3. При необходимости в меню "Порядок чтения документа" выберите порядок чтения документа.

    • Слева направо

    • Справа налево

    Примечание. Этим порядком также определяется сторона переплета документа. При выборе Слева направо сторона переплета будет на левой стороне буклета.

  4. Нажмите Печать.
