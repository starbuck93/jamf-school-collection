Utilisation de la fonction Comptabilité des travaux

Cette fonction permet d'effectuer le suivi des travaux d'impression en les associant à un ID ou nom d'utilisateur, à un code de compte et à un code de service. En associant aux travaux d'impression des informations de compte spécifiques, vous pouvez appliquer des quotas sur les travaux d'impression envoyés par les utilisateurs. Pour appliquer des quotas aux utilisateurs, installez Device Quotas sur l'imprimante. Pour obtenir plus d'informations, reportez-vous au Guide de l'administrateur de Device Quotas.

  1. Dans le menu des options, sélectionnez Compta travaux >Activer le suivi des travaux d'impression.

  2. Saisissez ou sélectionnez un ID ou nom d'utilisateur, un code de compte et un code de service.

    Remarque : Le nom de l'utilisateur connecté est utilisé par défaut. Pour créer un nom personnalisé, décochez Utiliser le nom de l'utilisateur connecté, puis saisissez le nom d'utilisateur.

  3. Cliquez sur Imprimer.
