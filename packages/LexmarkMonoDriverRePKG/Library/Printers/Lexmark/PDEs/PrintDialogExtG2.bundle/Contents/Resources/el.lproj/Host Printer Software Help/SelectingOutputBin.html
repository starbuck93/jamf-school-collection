Επιλογή δίσκου εξόδου

Σημείωση: Βεβαιωθείτε ότι ο εκτυπωτής σας υποστηρίζει προαιρετικούς δίσκους, προκειμένου να χρησιμοποιήσετε διαφορετικούς δίσκους από αυτούς που συνόδευαν αρχικά τον εκτυπωτή σας. Για περισσότερες πληροφορίες, ανατρέξτε στον Οδηγό χρήσης του εκτυπωτή.

  1. Στο μενού Επιλογές, επιλέξτε Τελείωμα.

  2. Στο μενού Δίσκος εξόδου, επιλέξτε το δίσκο που θέλετε να χρησιμοποιήσετε.

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Βασικός δίσκος—Χρησιμοποιεί το βασικό δίσκο του εκτυπωτή.

    • Δίσκος [x]—Χρησιμοποιεί έναν προαιρετικό δίσκο που διατίθεται για τον εκτυπωτή σας.

  3. Κάντε κλικ στην επιλογή Εκτύπωση.
