使用「進階版面配置」功能

此功能可讓您將文件列印成小冊子或海報。

  1. 在選項功能表中，選取進階版面配置。

  2. 在「版面配置」功能表中，選取下列其中一項：

    • 一般—不要將版面配置選項套用到列印工作。

    • 小冊子—將文件列印成小冊子。

    • 海報—使用縮版併印放大文件，將文件列印成海報。

  3. 如有必要，請在「文件讀取次序」功能表中，選取文件的讀取方式。

    • 由左至右

    • 由右至左

    請注意： 此次序也可決定小冊子的裝訂邊。若是選取了由左至右 ，裝訂邊是在小冊子的左邊。

  4. 按一下列印。
