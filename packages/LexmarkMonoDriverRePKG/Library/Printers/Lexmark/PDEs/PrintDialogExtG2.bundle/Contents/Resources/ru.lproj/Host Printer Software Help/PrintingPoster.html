Печать плаката

Данная функция позволяет увеличить распечатываемый документ в несколько раз от размера оригинала. В зависимости от выбранного размера плаката документ делится на равное количество частей, и каждая часть печатается на целом листе бумаги. После печати всех частей обрежьте белые поля вокруг каждой страницы и составьте из отдельных листов плакат.

  1. В меню параметров выберите Расширенный макет.

  2. В меню "Макет" выберите Плакат.

  3. В меню "Формат" выберите размер плаката.

  4. В меню "Наложение между страницами" выберите количество наложений между страницами. Данный параметр позволяет оставлять свободное пространство, благодаря которому страницы можно сложить вместе, не закрывая части изображения.

  5. При необходимости выберите Печать меток обрезки для печати линий разметки обрезки каждой части плаката.

  6. Нажмите Печать.
