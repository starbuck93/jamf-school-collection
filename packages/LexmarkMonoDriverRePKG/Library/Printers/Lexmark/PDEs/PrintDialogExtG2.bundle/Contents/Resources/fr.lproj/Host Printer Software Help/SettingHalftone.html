Définition des demi-tons

Cette fonction permet de sélectionner des linéatures demi-tons pour les travaux d'impression.

  1. Dans le menu des options, sélectionnez Qualité.

  2. Dans le menu Demi-tons, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Normal : utilise des linéatures demi-tons standard.

    • Détail : utiliser des linéatures demi-tons conçues pour les lignes de couleur fines et le texte plutôt que l'uniformité et le lissage.

  3. Cliquez sur Imprimer.
