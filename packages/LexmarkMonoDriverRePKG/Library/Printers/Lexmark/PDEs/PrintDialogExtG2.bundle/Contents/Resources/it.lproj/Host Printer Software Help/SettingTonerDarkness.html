Impostazione dell'intensità del toner

Questa funzione consente di schiarire o scurire gli output di stampa.

  1. Nel menu Opzioni, selezionare Qualità.

  2. Nel menu Intensità toner, selezionare una tra le seguenti opzioni:

    • Usa impostazione stampante: consente di utilizzare le impostazioni specificate nella stampante.

    • Manuale: personalizza l'intensità del toner della stampa.

  3. Utilizzare il dispositivo di scorrimento per regolare i valori di intensità toner.

  4. Fare clic su Stampa.
