Festlegen des Tonerauftrags

Mit dieser Funktion können Sie den Ausdruck heller oder dunkler gestalten.

  1. Wählen Sie im Options-Menü die Option Qualität.

  2. Wählen Sie im Menü "Tonerauftrag" eine der folgenden Optionen:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Manuell: Passt den Tonerauftrag der Druckausgabe an.

  3. Passen Sie den Tonerauftragswert mithilfe des Schiebereglers an.

  4. Klicken Sie auf Drucken.
