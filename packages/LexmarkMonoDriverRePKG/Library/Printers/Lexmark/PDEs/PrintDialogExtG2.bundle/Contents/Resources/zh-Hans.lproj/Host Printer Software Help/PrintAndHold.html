使用“打印并挂起”特性

此特性让您以几种方式将您的打印作业保存在打印机内存中。

注意： 如需有关释放挂起作业的更多信息，请参阅打印机用户指南或您用于释放作业的应用程序的管理员指南。

  1. 在选项菜单中，选择打印并挂起。

  2. 选择如何打印作业。

    • 保密打印—使用 4 位数的个人身份识别号码（PIN 码）来密码保护保密打印作业。

    • 保留打印—将打印作业保存在打印机内存中用于以后打印。

    • 校验打印—打印多副本打印作业的第一份副本以作检验。剩余的副本被挂起，直到它们被释放以打印或者被取消。

    • 重复打印—立即打印作业并将一份副本保存在打印机内存中，以便以后打印更多的副本。

  3. 如果需要，请更改关联到打印作业的用户名。

    注意： 默认情况下，使用已登录的用户名。如需创建自定义名称，请清除使用已登录的用户名，然后键入用户名。

  4. 如果选择保密打印，请输入与打印作业相关联的 PIN 码。

  5. 单击打印。
