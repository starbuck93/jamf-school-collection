Réglage de l'équilibre des couleurs

Cette fonction permet de régler la couleur du travail d'impression en augmentant ou en diminuant individuellement la quantité de toner utilisé pour chaque plan de couleur.

Remarque : Cette fonction est disponible uniquement pour les imprimantes couleur.

  1. Dans le menu des options, sélectionnez Couleur, puis cliquez sur Equilibre des couleurs.

  2. Dans le menu Equilibre des couleurs, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Manuel : personnalise la quantité de toner cyan, magenta, jaune et noir utilisé dans le travail d'impression.

  3. Si l'option Manuel est sélectionnée, utilisez les curseurs pour régler les paramètres suivants :

    • Rouge : ajuste l'équilibre entre le rouge et le cyan.

    • Vert : ajuste l'équilibre entre le vert et le magenta.

    • Bleu : ajuste l'équilibre entre le bleu et le jaune.

    • Blanc : ajuste l'équilibre entre le blanc et le noir.

  4. Cliquez sur Imprimer.
