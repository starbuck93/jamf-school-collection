Réglage de l'intensité du toner

Cette fonction permet d'éclaircir ou d'assombrir les travaux d'impression.

  1. Dans le menu des options, sélectionnez Qualité.

  2. Dans le menu Intensité du toner, sélectionnez l'une des options suivantes :

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Manuel : personnalise l'intensité du toner du travail d'impression.

  3. Vous pouvez régler la valeur d'intensité du toner à l'aide du curseur.

  4. Cliquez sur Imprimer.
