设置颜色修正

此特性调整打印的颜色以匹配其他输出设备或标准颜色系统的颜色。

注意： 此特性仅在彩色打印机中可用。

  1. 在选项菜单中，选择颜色，然后单击颜色修正。

  2. 在“颜色修正”菜单中，选择下列选项之一：

    • 使用打印机设置—使用打印机中指定的设置。

    • 关—禁用颜色修正。

    • 自动—将不同的颜色转换表应用到打印页面上的每一个对象。

    • 手动—定制应用到打印页面上每个对象的颜色转换表，并启用 RGB 和 CMYK 颜色修正设置。

    注意： 使用的颜色转换表取决于对象类型以及如何指定每个对象的颜色。

  3. 如果选择手动，请为每一个 RGB 和 CMYK 颜色修正设置选择一个选项。

    下列选项可用于图像、文本和图形 RGB 颜色修正：

    • 使用打印机设置—使用打印机中指定的设置。

    • 关—不应用颜色转换。

    • sRGB 显示—使打印的图像看起来与计算机显示器上的显示相像。此设置建议在打印图形时使用。

    • sRGB 逼真—使打印的图像比计算机显示器上的显示更饱和。此设置建议在打印商业图形和文本时使用。

    • 显示 - 纯黑色—仅使用黑色碳粉打印中性灰度颜色。

    • 逼真—使打印的图像更明亮、更饱和。

    下列选项可用于 CMYK 颜色修正：

    • 使用打印机设置—使用打印机中指定的设置。

    • 关—不应用颜色转换。

    • 逼真 CMYK—使用轮转胶印出版参数说明（SWOP）颜色转换表打印图像。

    • Euro CMYK—使用 Euroscale 颜色转换表打印图像。

    • US CMYK—当使用 U.S. CMYK 颜色转换表时，打印的图像更饱和。

  4. 单击打印。
