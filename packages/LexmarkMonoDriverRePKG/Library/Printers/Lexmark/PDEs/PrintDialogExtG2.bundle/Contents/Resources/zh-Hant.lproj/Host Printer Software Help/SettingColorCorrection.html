設定色彩修正

此功能可調整列印色彩，使其符合其他輸出裝置或標準色系中的色彩。

請注意： 此功能只適用於彩色印表機。

  1. 在選項功能表中，選取彩色，然後按一下色彩修正。

  2. 在「色彩修正」功能表中，選取下列其中一項：

    • 使用印表機設定—使用印表機中指定的設定值。

    • 關閉—停用色彩修正。

    • 自動—會在列印頁的每個物件上套用不同的色彩轉換表。

    • 手動—自訂套用至列印頁上每個物件的色彩轉換表，然後啟用 RGB 和 CMYK 色彩修正設定。

    請注意： 所使用的色彩轉換表，需視物件類型及各物件色彩的指定方式而定。

  3. 若是選取了手動 ，請為每個 RGB 和 CMYK 色彩修正設定選取選項。

    下列選項適用於影像、文字及圖形 RGB 色彩修正：

    • 使用印表機設定—使用印表機中指定的設定值。

    • 關閉—不套用色彩轉換。

    • sRGB 顯示—依照電腦螢幕所顯示的影像外觀來列印影像。列印圖形時，建議使用此設定。

    • sRGB 鮮明色彩—以比電腦螢幕所顯示的影像更加飽和的方式來列印影像。列印商業圖形和文字時，建議使用此設定。

    • 顯示 - 全黑—僅使用黑色碳粉列印中性灰色。

    • 鮮明色彩—以更加明亮及飽和的方式列印影像。

    下列選項適用於 CMYK 色彩修正：

    • 使用印表機設定—使用印表機中指定的設定值。

    • 關閉—不套用色彩轉換。

    • Vivid CMYK（鮮明色彩 CMYK）—使用 Web 平版印刷排版規格 (SWOP) 色彩轉換表列印影像。

    • Euro CMYK（歐式 CMYK）—使用 Euroscale 色彩轉換表列印影像。

    • US CMYK（美式 CMYK）—使用「美式 CMYK」色彩轉換表時，可以更加飽滿的方式列印影像。

  4. 按一下列印。
