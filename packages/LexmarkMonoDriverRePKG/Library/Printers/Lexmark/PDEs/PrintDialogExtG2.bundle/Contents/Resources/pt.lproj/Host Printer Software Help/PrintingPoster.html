Impressão de um cartaz

Este recurso permite ampliar várias vezes a saída impressa em relação ao tamanho original. Dependendo do tamanho do cartaz selecionado, o documento é dividido em um número igual de seções e cada seção é impressa em uma folha de papel inteira. Depois que todas as seções são impressas, apare os espaços extra em branco ao redor de cada página e junte as folhas para formar um cartaz.

  1. No menu de opções, selecione Layout avançado.

  2. No menu Layout, selecione Cartaz.

  3. No menu Tamanho, selecione o tamanho do cartaz.

  4. No menu Sobreposição de páginas, selecione a quantidade de sobreposição entre as páginas. Esta definição permite criar um espaço para juntar as páginas sem cobrir uma parte da imagem.

  5. Se necessário, selecione Imprimir marcas de corte para imprimir uma linha que serve como guia para aparar o excesso de papel de cada seção do cartaz.

  6. Clique em Imprimir.
