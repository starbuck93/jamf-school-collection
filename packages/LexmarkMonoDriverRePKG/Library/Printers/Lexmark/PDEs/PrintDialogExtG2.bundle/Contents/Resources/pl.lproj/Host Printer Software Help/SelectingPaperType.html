Wybieranie typu papieru

  1. W menu opcji wybierz Papier.

  2. W menu Typ papieru wybierz typ papieru, którego chcesz użyć.

    Uwagi:
    • Przed użyciem niestandardowych typów papieru skonfiguruj je w drukarce. Więcej informacji można znaleźć w Podręczniku użytkownika drukarki.
    • Można użyć ustawień drukarki.

  3. Kliknij przycisk Print (Drukuj).
