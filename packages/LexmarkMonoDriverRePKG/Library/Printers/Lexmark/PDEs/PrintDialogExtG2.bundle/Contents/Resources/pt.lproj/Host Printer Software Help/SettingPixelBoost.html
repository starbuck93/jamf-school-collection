Configuração do aumento de pixel

Este recurso aprimora os detalhes da imagem e do texto do documento impresso aumentando levemente a densidade de pixel de acordo com a definição selecionada.

Nota: Essa opção está disponível somente para uso com impressoras monocromáticas.

  1. No menu de opções, selecione Qualidade.

  2. No menu Aumento de pixels, selecione uma das opções a seguir:

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Desativar– Desabilita o aumento de pixel.

    • Fontes– Escurece o texto no documento impresso. Essa definição afeta somente os dados de fontes.

    • Horizontalmente- Escurece as linhas horizontais de imagem e texto no documento impresso.

    • Verticalmente- Escurece as linhas verticais de imagem e texto no documento impresso.

    • Ambas as direções- Escurece as linhas horizontais e verticais de imagem e texto no documento impresso.

  3. Clique em Imprimir.
