Εκτύπωση διαχωριστικής σελίδας

Χρησιμοποιήστε αυτήν τη λειτουργία, για να διαχωρίσετε ένα έγγραφο από ένα άλλο κατά την εκτύπωση πολλαπλών εγγράφων ή αντιγράφων του εγγράφου σε μία εργασία εκτύπωσης.

  1. Στο μενού Επιλογές, επιλέξτε Τελείωμα.

  2. Στο μενού Προέλευση διαχωριστικού, επιλέξτε τη θήκη της διαχωριστικής σελίδας.

  3. Στο μενού Διαχωριστική σελίδα, επιλέξτε πότε θα εκτυπώνεται μια διαχωριστική σελίδα.

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Καμία—Να μην εκτυπώνεται διαχωριστική σελίδα.

    • Ανάμεσα σε εργασίες—Εκτυπώνει μια διαχωριστική σελίδα μετά την ολοκλήρωση της εκτύπωσης μιας εργασίας εκτύπωσης.

    • Ανάμεσα σε αντίγραφα—Εκτυπώνει μια διαχωριστική σελίδα μετά την ολοκλήρωση της εκτύπωσης ενός αντιγράφου.

  4. Κάντε κλικ στην επιλογή Εκτύπωση.
