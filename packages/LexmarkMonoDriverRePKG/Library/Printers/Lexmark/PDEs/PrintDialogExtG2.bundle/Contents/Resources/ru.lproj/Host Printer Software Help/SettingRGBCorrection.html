Настройка коррекции RGB

Эта функция позволяет регулировать яркость, контрастность и насыщенность распечатываемых документов.

Примечание. Такая функция есть только в цветных принтерах.

  1. В меню параметров выберите Цвет, затем нажмите Коррекция RGB.

  2. В меню "Коррекция RGB" выберите одно из следующего:

    • Использовать настройки принтера — Использование настроек, заданных на принтере.

    • Вручную — настройка яркости, контрастности и насыщенности распечатываемых документов.

  3. При выборе Вручную ползунками регулируются следующие параметры:

    • Яркость — Настройка яркости распечатываемых документов.

    • Контрастность— Настройка контрастности распечатываемых документов.

    • Насыщенность — Настройка четкости и интенсивности цвета распечатываемых документов.

  4. Нажмите Печать.
