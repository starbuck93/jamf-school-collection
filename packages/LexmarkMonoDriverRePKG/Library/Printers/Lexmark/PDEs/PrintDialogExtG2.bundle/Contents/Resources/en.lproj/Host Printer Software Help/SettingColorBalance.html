Setting the color balance

This feature lets you adjust the color of printed outputs by individually increasing or decreasing the amount of toner used for each color plane.

Note: This feature is available only in color printers.

  1. In the options menu, select Color, and then click Color Balance.

  2. In the Color Balance menu, select one of the following:

    • Use Printer Setting—Use the setting that is specified in the printer.

    • Manual—Customize the amount of cyan, magenta, yellow, and black toners used in the printed output.

  3. If Manual is selected, then use the sliders to adjust the following settings:

    • Red—Adjust the color balance between red and cyan.

    • Green—Adjust the color balance between green and magenta.

    • Blue—Adjust the color balance between blue and yellow.

    • White—Adjust the color balance between white and black.

  4. Click Print.
