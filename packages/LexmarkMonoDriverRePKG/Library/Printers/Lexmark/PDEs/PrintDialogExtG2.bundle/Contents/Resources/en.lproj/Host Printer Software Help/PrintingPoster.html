Printing a poster

This feature lets you enlarge a printed output to several times the original size. Depending on the poster size selected, the document is divided into an equal number of sections, each of which prints on a full sheet of paper. After all sections are printed, trim the extra white spaces around each page and join the individual sheets to create a poster.

  1. In the options menu, select Advanced Layout.

  2. In the Layout menu, select Poster.

  3. In the Size menu, select the size of the poster.

  4. In the Overlap Between Pages menu, select the amount of overlap between pages. This setting lets you create a space where you can attach the pages together without covering a portion of the image.

  5. If necessary, select Print Crop Marks to print a line that serves as a guide for trimming excess paper from each section of the poster.

  6. Click Print.
