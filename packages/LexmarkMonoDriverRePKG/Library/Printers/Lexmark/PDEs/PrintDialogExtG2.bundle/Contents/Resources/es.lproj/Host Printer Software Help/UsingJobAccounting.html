Uso de la función Recuento de trabajos

Esta función permite realizar un recuento de los trabajos de impresión al asociarlos con el nombre o ID de usuario, el código de cuenta y el código de departamento. Al asociar los trabajos de impresión a la información de cuenta específica, también podrá imponer cuotas para los trabajos de impresión que envíen los usuarios. Para imponer cuotas a los usuarios, instale Cuotas de dispositivos en la impresora. Para obtener más información, consulte la Guía del administrador de Cuotas de dispositivos.

  1. En el menú Opciones, seleccione Recuento de trabajos >Activar recuento de trabajos.

  2. Escriba o selecciona un nombre de usuario o ID, un código de cuenta y un código de departamento.

    Nota: De forma predeterminada, se utiliza el nombre del usuario. Para crear un nombre personalizado, borre Utilizar nombre de usuario con sesión iniciada e introduzca el nombre del usuario.

  3. Haga clic en Imprimir.
