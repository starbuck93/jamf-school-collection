Εκτύπωση με τη λειτουργία μετατόπισης

Αυτή η λειτουργία στοιβάζει αντίγραφα σε κλιμακωτά σύνολα στον επιλεγμένο δίσκο.

Σημείωση: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη μονάδα μετατόπισης στον εκτυπωτή σας. Για περισσότερες πληροφορίες, ανατρέξτε στον Οδηγό χρήσης του εκτυπωτή.

  1. Στο μενού Επιλογές, επιλέξτε Τελείωμα.

  2. Στο μενού Μετατόπιση σελίδων, επιλέξτε μία από τις παρακάτω ρυθμίσεις:

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Ενεργοποίηση—Ενεργοποιεί τη λειτουργία μετατόπισης.

    • Απενεργοποίηση—Απενεργοποιεί τη λειτουργία μετατόπισης.

  3. Κάντε κλικ στην επιλογή Εκτύπωση.
