포스터 인쇄

이 기능을 사용하면 원래 크기의 몇 배로 인쇄 출력물을 확대할 수 있습니다. 선택된 포스터 크기에 따라 문서는 동일한 수의 섹션으로 나뉘며, 한 섹션이 용지 하나에 인쇄됩니다. 모든 섹션을 인쇄한 후 각 페이지 주위의 여백을 정리하고 각 장을 접합하여 포스터를 만듭니다.

  1. 옵션 메뉴에서 고급 레이아웃을 선택합니다.

  2. 레이아웃 메뉴에서 포스터를 선택합니다.

  3. 크기 메뉴에서 포스터의 크기를 선택합니다.

  4. 페이지 간 겹치기 메뉴에서 페이지 간 겹치기의 정도를 선택합니다. 이 설정을 사용하면 이미지의 일부를 덮지 않고 페이지들을 함께 연결할 수 있는 공간을 만들 수 있습니다.

  5. 필요한 경우 포스터의 각 섹션의 초과 용지를 정리하기 위한 가이드의 역할을 하는 라인을 인쇄하기 위해 절단선 인쇄를 선택합니다.

  6. 인쇄를 클릭합니다.
