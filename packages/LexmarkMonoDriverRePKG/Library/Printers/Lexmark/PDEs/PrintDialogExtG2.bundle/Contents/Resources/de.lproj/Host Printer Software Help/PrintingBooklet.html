Drucken einer Broschüre

Mit dieser Funktion können Sie mehrseitige Dokumente ausdrucken, die dann zu einer Broschüre gefaltet werden können.

  1. Wählen Sie im Options-Menü die Option Erweitertes Layout.

  2. Wählen Sie im Menü "Erweitertes Layout" die Option Broschüre.

  3. Unter "Blatt pro Bündel" können Sie die Anzahl der Seiten pro Faltbündel festlegen.

    Hinweis: Wenn Sie unter dieser Option einen passenden Wert eingeben, lässt sich das Bündel gut falten und die Blattkanten können gleichmäßig ausgerichtet werden.

  4. Wählen Sie ggf. Ursprüngliches Seitenformat beibehalten, um die Größe der gedruckten Broschüre zu ändern.

    Hinweise:
    • Wenn Ursprüngliches Seitenformat beibehalten ausgewählt ist, hat jede Seite der fertigen Broschüre dasselbe Format wie das Originaldokument. Wenn das Originaldokument z.B. eine Größe von 8,5 x 11 Zoll hat, wird die Broschüre mit dem Papierformat 11 x 17 Zoll gedruckt. Bei Verwendung des Papierformats 11 x 17 Zoll hat der Ausdruck dieselben Maße wie das Originaldokument. Diese Einstellung eignet sich am besten für Originaldokumente im Format Statement, Letter, A5, A4 und B5.
    • Wenn Ursprüngliches Seitenformat beibehalten nicht ausgewählt ist, werden die einzelnen Seiten der Broschüre auf das Papierformat des Originaldokuments runterskaliert.

  5. Um Deckblätter hinzuzufügen, wählen Sie Broschürendeckblatt einschließen und danach mindestens eine der folgenden Optionen:

    • Vorderes Deckblatt außen bedrucken

    • Vorderes Deckblatt innen bedrucken

    • Hinteres Deckblatt außen bedrucken

    • Hinteres Deckblatt innen bedrucken

  6. Wählen Sie im Menü "Deckblatt-Papiertyp" die Papiersorte für das Broschürendeckblatt aus.

  7. Wählen Sie im Menü "Deckblatt-Papierquelle" das Papierfach für das Broschürendeckblatt aus.

  8. Klicken Sie auf Drucken.
