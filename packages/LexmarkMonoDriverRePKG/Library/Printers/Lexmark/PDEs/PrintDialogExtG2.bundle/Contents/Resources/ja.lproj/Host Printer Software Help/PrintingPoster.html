ポスターを印刷する

この機能を使うと、標準サイズの数倍に拡大したサイズの印刷を行うことができます。 選択したポスターのサイズに応じて、文書は選択した数字と同数のセクションに分割され、各セクションが用紙に全面印刷されます。 すべてのセクションを印刷した後、各用紙の周囲の余白を切り取ってつなぎ合わせることで、1 枚のポスターを作成できます。

  1. ［オプション］メニューで、［高度なレイアウト］を選択します。

  2. ［レイアウト］メニューで、［ポスター］を選択します。

  3. ［サイズ］メニューで、ポスターのサイズを選択します。

  4. ［ページ間の重複］メニューで、ページが重なる幅を選択します。 これにより、画像部分を欠落させずにページを接着するためののりしろが作成されます。

  5. 必要に応じて、［トンボを印刷］を選択してポスターの各セクションで切り取る余白の目印にする切り抜き線を印刷します。

  6. ［印刷］をクリックします。
