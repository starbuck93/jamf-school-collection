Using the Advanced Layout feature

This feature lets you print documents as booklets or posters.

  1. In the options menu, select Advanced Layout.

  2. In the Layout menu, select one of the following:

    • Normal—No layout options are applied to the print job.

    • Booklet—Print a document as a booklet.

    • Poster—Print a document as a poster by enlarging it using multiple pages.

  3. If necessary, in the Document Reading Order menu, select how the document is read.

    • Left to right

    • Right to left

    Note: This order also determines the binding edge of the booklet. If Left to right is selected, then the binding edge is on the left side of the booklet.

  4. Click Print.
