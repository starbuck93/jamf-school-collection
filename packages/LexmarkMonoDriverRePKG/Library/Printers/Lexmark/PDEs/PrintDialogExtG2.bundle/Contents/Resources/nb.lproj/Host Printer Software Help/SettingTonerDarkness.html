Angi tonermørkhet

Med denne funksjonen kan du gjøre utskriften lysere eller mørkere.

  1. Velg Kvalitet på menyen Alternativer.

  2. Velg ett av følgende på menyen Tonermørkhet:

    • Bruk skriverinnstilling: Bruker innstillingen som er angitt i skriveren.

    • Manuell: Justerer mørkheten for utskriften.

  3. Bruk glidebryteren for å justere tonermørkhetsverdiene.

  4. Klikk på Skriv ut.
