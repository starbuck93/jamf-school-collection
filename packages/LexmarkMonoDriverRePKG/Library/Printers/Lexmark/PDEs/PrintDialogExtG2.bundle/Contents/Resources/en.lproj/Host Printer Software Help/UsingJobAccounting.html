Using the Job Accounting feature

This feature lets you track print jobs by associating them with a user name or ID, an account code, and a department code. By associating print jobs with specific account information, you can enforce quotas on the print jobs that users can send. To enforce quotas on users, install Device Quotas on the printer. For more information, see the Device Quotas Administrator’s Guide.

  1. In the options menu, select Job Accounting > Enable job accounting.

  2. Type or select a user name or ID, an account code, and a department code.

    Note: By default, the logged‑in user name is used. To create a custom name, clear Use logged‑in user name, and then type the user name.

  3. Click Print.
