소책자 인쇄

이 기능을 사용하면 소책자로 접을 수 있는 여러 페이지의 문서를 인쇄할 수 있습니다.

  1. 옵션 메뉴에서 고급 레이아웃을 선택합니다.

  2. 레이아웃 메뉴에서 소책자를 선택합니다.

  3. 번들당 용지 매수 메뉴에서 각 접힌 번들에 포함되는 페이지의 수를 지정합니다.

    참고: 이 옵션을 적합한 숫자로 설정하면 번들을 쉽게 접을 수 있고 페이지 가장자리 줄이 고르게 맞춰집니다.

  4. 필요한 경우 원본 페이지 크기 유지를 선택하여 인쇄 소책자의 크기를 변경합니다.

    참고:
    • 원본 페이지 크기 유지를 선택한 경우 소책자의 각 페이지는 원본 문서와 동일한 크기입니다. 예를 들어 원래의 문서가 8.5 x 11인치인 경우 소책자는 11 x 17인치의 용지에 인쇄됩니다. 11 x 17인치 용지에 인쇄하면 원본 문서의 치수가 유지됩니다. 이 설정은 Statement, Letter, A5, A4 또는 B5 용지 크기의 원본 문서에 가장 적합합니다.
    • 원본 페이지 크기 유지를 선택하지 않은 경우 소책자의 개별 페이지는 원본 문서의 용지 크기로 축소됩니다.

  5. 필요한 경우 소책자 표지 포함을 선택하여 표지 페이지의 추가를 활성화한 후 다음 중에서 하나 이상의 옵션을 선택합니다.

    • 앞 표지 바깥 인쇄

    • 앞 표지 안 인쇄

    • 뒷 표지 바깥 인쇄

    • 뒷 표지 안 인쇄

  6. 표지 용지 유형 메뉴에서 소책자 표지의 용지 유형을 선택합니다.

  7. 표지 용지 급지대 메뉴에서 소책자 표지에 사용할 용지함을 선택합니다.

  8. 인쇄를 클릭합니다.
