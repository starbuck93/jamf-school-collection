Configuração de meio-tom

Este recurso permite selecionar telas de meio-tom alternativas para a saída impressa.

  1. No menu de opções, selecione Qualidade.

  2. No menu Meio-tom, selecione uma das opções a seguir:

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Normal–Use telas de sombreados padrão.

    • Detalhes–Use telas de sombreados alternativos projetadas para linhas de cores finas e texto em vez de uniformidade e lisura.

  3. Clique em Imprimir.
