고급 레이아웃 기능 사용

이 기능을 사용하면 문서를 소책자나 포스터로 인쇄할 수 있습니다.

  1. 옵션 메뉴에서 고급 레이아웃을 선택합니다.

  2. 레이아웃 메뉴에서 다음 중 하나를 선택합니다.

    • 일반 - 레이아웃 옵션이 인쇄 작업에 적용되지 않습니다.

    • 소책자—문서를 소책자로 인쇄합니다.

    • 포스터 - 여러 페이지로 확대해서 문서를 포스터로 인쇄합니다.

  3. 필요한 경우 문서 읽기 순서 메뉴에서 문서를 읽는 방식을 선택합니다.

    • 왼쪽에서 오른쪽

    • 오른쪽에서 왼쪽

    참고: 이 순서는 소책자의 바인딩 가장자리도 결정합니다. 왼쪽에서 오른쪽을 선택한 경우 바인딩 가장자리는 소책자의 왼쪽입니다.

  4. 인쇄를 클릭합니다.
