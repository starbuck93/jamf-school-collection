Configuração das correções de cores

Este recurso ajusta as cores impressas para que elas correspondam às cores de outros dispositivos de saída ou sistemas de cores padrão.

Nota: Este recurso está disponível somente em impressoras coloridas.

  1. No menu de opções, selecione Core, em seguida, clique em Correção de cores.

  2. No menu Correção de cores, selecione uma das seguintes opções:

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Desativar— Desativa a correção de cores.

    • Automático-Aplica uma diferente tabela de conversão de cores a cada objeto da página impressa.

    • Manual-Personaliza a tabela de conversão de cores que é aplicada a cada objeto da página impressa e ativa as configurações de correção de cores RGB e CMYK.

    Nota: A tabela de conversão de cores utilizada depende do tipo de objeto e de como é especificada a cor de cada objeto.

  3. Se a opção Manual for selecionada, selecione então uma opção para cada configuração de correção de cores RGB e CMYK.

    As seguintes opções estão disponíveis para correção de cores RGB em imagens, textos e gráficos:

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Desligado-Não aplica conversão de cores.

    • Exibir sRGB-Imprime as imagens com a aparência de como elas são exibidas em um monitor de computador. Esta configuração é recomendada ao imprimir gráficos.

    • Cores vivas sRGB-Imprime as imagens com a aparência mais saturada de como elas serão exibidas em um monitor de computador. Esta configuração é recomendada ao imprimir gráficos e textos comerciais.

    • Exibir - preto real-Imprime cores cinza neutro usando apenas o toner preto.

    • Cores vivas- Imprime imagens mais brilhantes e saturadas.

    As opções a seguir estão disponíveis para correção de cores CMYK:

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Desligado-Não aplica conversão de cores.

    • Cores vivas CMYK-Imprime imagens por meio de especificações da tabela de conversão de cores de publicações offset da Web (SWOP).

    • Euro CMYK-Imprime imagens usando a tabela de conversão de cores Euroscale.

    • US CMYK- Imprime imagens mais saturadas ao usar as configurações padrão dos EUA Tabela de conversão de cores CMYK.

  4. Clique em Imprimir.
