Festlegen der Druckauflösung

Mit dieser Funktion können Sie die Auflösung des Ausdrucks in dpi (dots per inch) oder CQ (color quality, Farbqualität) festlegen. Je höher die Druckauflösung, umso höher ist die Qualität bzw. Schärfe des Ausdrucks.

  1. Wählen Sie im Options-Menü die Option Qualität.

  2. Wählen Sie im Menü "Auflösung" eine Auflösung aus.

    Hinweis: Die Optionen können je nach Drucker variieren. Weitere Informationen finden Sie im Benutzerhandbuch des Druckers.

  3. Klicken Sie auf Drucken.
