Perforation de documents imprimés

Remarque : Assurez-vous qu'une unité de finition de perforation est installée sur votre imprimante. Pour plus d'informations, reportez-vous au Guide de l'utilisateur de l'imprimante.

  1. Dans le menu des options, sélectionnez Finition.

  2. Dans le menu Perforation, sélectionnez le nombre de trous que vous souhaitez utiliser.

    Remarque : Les trous sont faits sur le bord gauche ou droit de la page.

    • Utiliser le paramètre de l'imprimante : applique le paramètre spécifié dans l'imprimante.

    • Désactivé : désactive la fonction de perforation.

    • 2 trous

    • 3 trous

    • 4 trous

  3. Cliquez sur Imprimer.
