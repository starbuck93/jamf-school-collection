Выбор типа бумаги

  1. В меню параметров выберите Бумага.

  2. В меню "Тип бумаги" выберите тип бумаги, которая будет использоваться.

    Примечания.
    • Перед использованием пользовательских типов бумаги следует выполнить соответствующую настройку принтера. Для получения дополнительной информации см. Руководство пользователя принтера.
    • Можно использовать настройки принтера.

  3. Нажмите Печать.
