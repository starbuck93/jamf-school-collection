Печать брошюры

Эта функция позволяет напечатать многостраничный документ в виде брошюры.

  1. В меню параметров выберите Расширенный макет.

  2. В меню "Макет" выберите Брошюра.

  3. В меню "Листов в пачке" укажите количество страниц, включаемых в одну пачку.

    Примечание. При задании соответствующего количества страниц пачка будет легче сгибаться, и края страниц будут располагаться ровнее.

  4. При необходимости выберите Сохранить формат исходной страницы, чтобы изменить формат распечатанной брошюры.

    Примечания.
    • При выборе Сохранить формат исходной страницы каждая страница брошюры будет того же формата, что и в оригинальном документе. Например, если формат оригинала 8,5 x 11", брошюра будет напечатана на бумаге формата 11 x 17". При печати на бумаге формата 11 x 17" сохраняются размеры оригинального документа. Данный параметр лучше всего использовать с форматами оригинала Statement, Letter, A5, A4, или B5.
    • Если параметр Сохранить формат исходной страницы не выбран, масштаб отдельных страниц брошюры изменяется до формата оригинала.

  5. При необходимости выберите Включая обложку брошюры для добавления страниц обложки, затем выберите одно или более значений:

    • Печатать на внешней передней обложке

    • Печатать на внутренней передней обложке

    • Печатать на внешней задней обложке

    • Печатать на внутренней задней обложке

  6. В меню "Тип бумаги обложки" выберите тип бумаги для обложки.

  7. В меню "Источник бумаги для обложки" выберите используемый для обложки лоток.

  8. Нажмите Печать.
