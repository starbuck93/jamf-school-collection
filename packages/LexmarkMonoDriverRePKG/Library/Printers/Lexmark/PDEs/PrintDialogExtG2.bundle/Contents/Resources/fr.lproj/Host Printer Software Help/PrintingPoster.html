Impression d'une affiche

Cette fonction permet d'agrandir un travail d'impression plusieurs fois par rapport au format d'origine. En fonction du format de poster sélectionné, votre document est divisé en un nombre identique de sections et chaque section est imprimée sur une feuille de papier complète. Lorsque toutes les sections sont imprimées, vous pouvez découper les zones blanches autour de chaque page et assembler les feuilles individuelles pour obtenir une affiche.

  1. Dans le menu des options, sélectionnez Mise en page avancée.

  2. Dans le menu Mise en page, sélectionnez Affiche.

  3. Dans le menu Format, sélectionnez le format de l'affiche.

  4. Dans le menu Recouvrement entre les pages, sélectionnez la valeur de recouvrement entre les pages. Ce paramètre vous permet de créer un espace grâce auquel vous pouvez assembler les pages sans recouvrir une partie de l'image.

  5. Si nécessaire, sélectionnez Imprimer les repères de coupe afin d'imprimer une ligne qui servira de guide pour découper l'excédent de papier dans chaque section de l'affiche.

  6. Cliquez sur Imprimer.
