Verwenden der Funktion "ColorSaver"

Diese Funktion verringert die vom Drucker verwendete Tonermenge beim Drucken von Farbbildern.

Hinweis: Diese Funktion steht nur auf Farbdruckern zur Verfügung.

  1. Wählen Sie im Options-Menü die Option Farbeaus und klicken Sie dann auf Farbe.

  2. Wählen Sie im Menü ColorSaverTM eine der folgenden Optionen aus:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Ein: Aktiviert ColorSaver.

    • Aus: Deaktiviert ColorSaver.

    Hinweis: Der Text wird mit der standardmäßigen Tonermenge gedruckt.

  3. Klicken Sie auf Drucken.
