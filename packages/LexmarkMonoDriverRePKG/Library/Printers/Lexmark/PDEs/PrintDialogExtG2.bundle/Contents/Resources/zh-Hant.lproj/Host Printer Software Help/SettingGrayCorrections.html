設定灰色修正

此功能可在進行黑白列印時調整文件的亮度和對比。

請注意： 此功能只適用於黑白印表機。

  1. 在選項功能表中，選取彩色。

  2. 在「灰色修正」功能表中，選取下列其中一項：

    • 使用印表機設定—使用印表機中指定的設定值。

    • 關閉—停用灰色調修正

    • 手動—自訂列印輸出的亮度及對比。

  3. 若是選取了手動，請使用滑桿調整下列設定：

    • 亮度—調整列印輸出的亮度。

    • 對比—調整列印輸出的對比。

  4. 按一下列印。
