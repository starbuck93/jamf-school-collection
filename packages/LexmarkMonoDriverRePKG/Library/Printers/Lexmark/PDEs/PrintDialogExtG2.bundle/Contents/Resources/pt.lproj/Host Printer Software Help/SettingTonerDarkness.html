Configuração da tonalidade do toner

Este recurso permite clarear ou escurecer as saídas impressas.

  1. No menu de opções, selecione Qualidade.

  2. No menu Tonalidade do toner, selecione uma das opções a seguir:

    • Usar Configurações da Impressora-Usa a configuração especificada na impressora.

    • Manual– Personaliza a tonalidade do toner da saída impressa.

  3. Use o controle deslizante para ajustar o valor de tonalidade do toner.

  4. Clique em Imprimir.
