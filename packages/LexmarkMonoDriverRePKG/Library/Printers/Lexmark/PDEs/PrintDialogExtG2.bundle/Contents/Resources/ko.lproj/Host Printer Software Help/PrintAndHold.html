“인쇄 및 대기” 기능 사용

이 기능을 사용하면 여러 가지 방법으로 인쇄 작업을 프린터 메모리에 저장할 수 있습니다.

참고: 보류 작업 해제에 관한 자세한 내용은 프린터 사용 설명서 또는 작업을 해제하는 데 사용하는 응용 프로그램의 관리자 설명서를 참조하십시오.

  1. 옵션 메뉴에서 인쇄 및 대기를 선택합니다.

  2. 작업 인쇄 방식을 선택합니다.

    • 기밀자료 인쇄—4자리 개인 식별 번호(PIN)를 사용하여 기밀 인쇄 작업을 비밀번호로 보호합니다.

    • 인쇄 예약—작업을 나중에 인쇄할 수 있도록 프린터 메모리에 저장합니다.

    • 인쇄 확인—확인을 위해 다중 복사 인쇄 작업의 첫 번째 사본을 인쇄합니다. 나머지 사본은 인쇄 작업을 릴리스하거나 취소할 때까지 보류됩니다.

    • 인쇄 반복—즉시 작업을 인쇄하고 나중에 여러 장을 인쇄할 수 있도록 프린터 메모리에 사본을 저장합니다.

  3. 필요한 경우 인쇄 작업과 관련된 사용자 이름을 변경합니다.

    참고: 기본적으로 로그인한 사용자 이름이 사용됩니다. 사용자 정의 이름을 만들려면 로그인 사용자 이름 사용을 지운 후 사용자 이름을 입력합니다.

  4. 기밀자료 인쇄를 선택한 경우 인쇄 작업과 관련된 PIN을 입력합니다.

  5. 인쇄를 클릭합니다.
