Использование функции "Отложенная печать"

Данная функция позволяет сохранять задания печати в памяти принтера различными способами.

Примечание. Подробнее о возобновлении печати отложенного задания см. в Руководстве пользователя или в Руководстве администратора приложения, используемого для возобновления задания.

  1. В меню параметров выберите Отложенная печать.

  2. Выберите способ печати задания.

    • Конфиденциальная печать — Конфиденциальная защищенная паролем печать заданий с использованием четырехзначного персонального идентификационного номера (ПИН).

    • Печать с резервированием — Позволяет сохранить задание на печать в памяти принтера и распечатать его позднее.

    • Печать с проверкой — При выполнении задания, предусматривающего печать нескольких копий, сначала распечатывается первая копия для проверки. Остальные копии не распечатываются и сохраняются в памяти до тех пор, пока они не будут распечатаны или отменены.

    • Печать с повтором — Печать задания сразу и сохранение ее в памяти принтера для печати дополнительных копий позже.

  3. При необходимости измените имя пользователя, связанное с заданием печати.

    Примечание. По умолчанию используется имя пользователя, под которым выполнен вход в систему. Для создания пользовательского имени очистите поле Использовать имя пользователя, вошедшего в систему, затем введите имя пользователя.

  4. При выборе Конфиденциальная печать введите ПИН-код этого задания печати.

  5. Нажмите Печать.
