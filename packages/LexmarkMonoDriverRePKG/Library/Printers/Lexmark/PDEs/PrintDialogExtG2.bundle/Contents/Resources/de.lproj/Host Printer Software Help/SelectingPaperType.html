Auswählen einer Papiersorte

  1. Wählen Sie im Options-Menü die Option Papier

  2. Wählen Sie im Menü "Papiersorte" die gewünschte Papierart aus.

    Hinweise:
    • Konfigurieren Sie den Drucken entsprechend, bevor Sie eine benutzerdefinierte Papiersorte verwenden. Weitere Informationen finden Sie im Benutzerhandbuch des Druckers.
    • Sie können die Druckereinstellungen verwenden.

  3. Klicken Sie auf Drucken.
