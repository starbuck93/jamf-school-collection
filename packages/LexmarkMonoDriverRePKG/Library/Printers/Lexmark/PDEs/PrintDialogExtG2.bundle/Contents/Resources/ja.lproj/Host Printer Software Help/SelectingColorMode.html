カラーモードを選択する

  1. ［オプション］メニューで、［カラー］を選択します。

  2. カラープリンタで、［カラー］をクリックします。

  3. ［カラーモード］メニューより、以下の中からいずれか1つを選択します。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • ［カラー］ - ドキュメントをカラーで印刷します。

      メモ： このオプションが使用できるのは、カラープリンタだけです。

    • ［モノクロ］ - ドキュメントをモノクロで印刷します。

  4. ［印刷］をクリックします。
