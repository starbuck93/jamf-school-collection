Setting RGB corrections

This feature lets you adjust the brightness, contrast, and saturation of the printed output.

Note: This feature is available only in color printers.

  1. In the options menu, select Color, and then click RGB Correction.

  2. In the RGB Correction menu, select one of the following:

    • Use Printer Setting—Use the setting that is specified in the printer.

    • Manual—Customize the brightness, contrast, and saturation of the printer output.

  3. If Manual is selected, then use the sliders to adjust the following settings:

    • Brightness—Adjust the brightness of the printed output.

    • Contrast—Adjust the contrast of the printed output.

    • Saturation—Adjust the vividness and color intensity of the printed output.

  4. Click Print.
