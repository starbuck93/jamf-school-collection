Drucken von Postern

Mit dieser Funktion können Sie einen Ausdruck auf das Mehrfache der Originalgröße vergrößern. Je nach ausgewählter Postergröße wird das Dokument in eine gerade Anzahl von Abschnitten aufgeteilt, und jeder Abschnitt wird auf einem einzelnen Blatt Papier gedruckt. Nach dem Druck aller Abschnitte können Sie den zusätzlichen weißen Rand an den einzelnen Seiten abschneiden und die Blätter zu einem Poster zusammenfügen.

  1. Wählen Sie im Options-Menü die Option Erweitertes Layout.

  2. Wählen Sie im Menü "Erweitertes Layout" die Option Poster.

  3. Wählen Sie im Menü "Größe" die Postergröße aus.

  4. Legen Sie im Menü "Seitenüberlappung" den Wert für die Seitenüberlappung fest. Auf diese Weise haben Sie genügend Platz, um die Seiten zusammenzufügen, ohne Teile des Bildes zu verdecken.

  5. Wählen Sie ggf. Zuschnittmarken drucken, um eine Linie zu drucken, anhand der Sie die einzelnen Posterabschnitte zuschneiden können.

  6. Klicken Sie auf Drucken.
