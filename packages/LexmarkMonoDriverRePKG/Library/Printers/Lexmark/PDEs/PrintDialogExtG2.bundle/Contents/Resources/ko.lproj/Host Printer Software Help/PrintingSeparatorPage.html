구분자 페이지 인쇄

여러 문서를 인쇄하거나 단일 인쇄 작업에서 문서의 여러 사본을 인쇄하는 경우 문서를 서로 분리하려면 이 기능을 사용합니다.

  1. 옵션 메뉴에서 마무리를 선택합니다.

  2. 간지 급지대 메뉴에서 구분자 페이지의 용지함을 선택합니다.

  3. 구분자 페이지 메뉴에서 구분자 페이지를 언제 인쇄할지 선택합니다.

    • 프린터 설정 사용—프린터에 지정된 설정을 사용합니다.

    • 없음—구분자 페이지를 인쇄하지 않습니다.

    • 작업 사이—인쇄 작업에서 인쇄를 마친 후 구분자 페이지를 인쇄합니다.

    • 사본 사이—인쇄 작업에서 1부 인쇄를 마친 후 구분자 페이지를 인쇄합니다.

  4. 인쇄를 클릭합니다.
