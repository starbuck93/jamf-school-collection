Festlegen der Halbtöne

Mit dieser Funktion können Sie alternative Halbtöne für den Ausdruck auswählen.

  1. Wählen Sie im Options-Menü die Option Qualität.

  2. Wählen Sie im Menü "Halbton" eine der folgenden Optionen aus:

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Normal: Es wird die Standard-Halbtonrasterung verwendet.

    • Details: Es werden alternative Halbtonrasterungen verwendet, die eher für dünne Farblinien und Text als für einheitliche und glatte Flächen gedacht sind.

  3. Klicken Sie auf Drucken.
