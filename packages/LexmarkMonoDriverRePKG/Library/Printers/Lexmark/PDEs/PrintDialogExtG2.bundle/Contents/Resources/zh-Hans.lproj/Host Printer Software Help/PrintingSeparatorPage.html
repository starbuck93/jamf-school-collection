打印分隔页

当在一个打印作业中打印多个文档或文档的多份副本时，使用此特性来将一个文档与另一个文档分开。

  1. 在选项菜单中，选择输出。

  2. 在“分隔页纸张来源”菜单中，选择分隔页的进纸匣。

  3. 在“分隔页”菜单中，选择何时打印分隔页。

    • 使用打印机设置—使用打印机中指定的设置。

    • 无—不打印分隔页。

    • 作业之间—在打印作业完成打印之后打印分隔页。

    • 副本之间—在打印作业的副本完成打印之后打印分隔页。

  4. 单击打印。
