Ställa in tonersvärta

Med den här funktionen kan du göra utskriften ljusare eller mörkare.

  1. Välj Kvalitet på alternativmenyn.

  2. Välj något av följande alternativ på menyn Tonersvärta:

    • Använd skrivarinställning – Använd inställningen som har angetts i skrivaren.

    • Manuell—Anpassa tonerns svärta i utskriften.

  3. Använd skjutreglaget för att justera värdet för tonersvärta.

  4. Klicka på Skriv ut.
