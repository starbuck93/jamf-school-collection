ColorSaver gebruiken

Deze functie vermindert de hoeveelheid toner die de printer gebruikt om afbeeldingen in kleur af te drukken.

Opmerking: Deze functie is alleen beschikbaar op kleurenprinters.

  1. Selecteer Kleur in het menu Opties en klik vervolgens op Kleur.

  2. Selecteer een van de volgende opties in het menu ColorSaverTM:

    • Printerinstellingen gebruiken: gebruikt de instelling die op de printer wordt opgegeven.

    • Aan: ColorSaver inschakelen.

    • Uit: ColorSaver uitschakelen.

    Opmerking: Tekst wordt afgedrukt met de standaardhoeveelheid toner.

  3. Klik op Afdrukken.
