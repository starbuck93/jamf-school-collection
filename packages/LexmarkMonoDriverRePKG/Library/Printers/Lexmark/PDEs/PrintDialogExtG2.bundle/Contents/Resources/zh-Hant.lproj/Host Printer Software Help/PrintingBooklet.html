列印小冊子

此功能可讓您列印可摺疊成小冊子的多頁文件。

  1. 在選項功能表中，選取進階版面配置。

  2. 在「版面配置」功能表中，選取小冊子。

  3. 在「每冊頁數」功能表中，指定每本摺好的小冊子所含的頁數。

    請注意： 將此選項設成適當的數目，不僅讓折疊小冊子變簡單，還能讓紙張邊緣排列整齊。

  4. 如有必要，請選取維持原始頁面尺寸 ，以變更列印小冊子的尺寸。

    請注意：
    • 若是選取了維持原始頁面尺寸，小冊子各頁面的大小均與原始文件相同。例如，若原始文件是 8.5 x 11 英吋，小冊子就會列印在 11 x 17‑英吋的紙張上。列印在 11 x 17‑英吋的紙張上，可保留原始文件的尺寸。此設定最適合使用於含有 Statement、Letter、A5、A4 或 B5 紙張尺寸的原始文件。
    • 若不是選取維持原始頁面尺寸，小冊子的個別頁面會縮小為原始文件的紙張尺寸。

  5. 如有必要，請選取包括小冊子封面，以新增封面，然後選取下列其中一或多個選項：

    • 列印在封面外頁上

    • 列印在封面內頁上

    • 列印在封底外頁上

    • 列印在封底內頁上

  6. 在「封面紙張類型」功能表中，選取小冊子封面的紙張類型。

  7. 在「封面紙張來源」功能表中，選取小冊子封面所使用的紙匣組件。

  8. 按一下列印。
