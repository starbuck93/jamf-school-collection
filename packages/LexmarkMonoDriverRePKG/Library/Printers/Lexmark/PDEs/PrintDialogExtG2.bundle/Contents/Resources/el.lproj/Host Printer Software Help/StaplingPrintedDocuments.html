Συρραφή εκτυπωμένων εγγράφων

Σημείωση: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη μονάδα συρραφής στον εκτυπωτή σας. Για περισσότερες πληροφορίες, ανατρέξτε στον Οδηγό χρήσης του εκτυπωτή.

  1. Στο μενού Επιλογές, επιλέξτε Τελείωμα.

  2. Στο μενού Συρραφή, επιλέξτε τον τρόπο συρραφής ενός εγγράφου.

    Οι παρακάτω επιλογές είναι διαθέσιμες για τη μονάδα συρραφής πολλαπλών θέσεων και τη μονάδα διάτρησης:

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Απενεργοποίηση—Απενεργοποιεί τη λειτουργία συρραφής.

    • Επάνω αριστερά—Τοποθέτηση συρραφής στην επάνω αριστερή γωνία των σελίδων.

    • Επάνω δεξιά—Τοποθέτηση συρραφής στην επάνω δεξιά γωνία των σελίδων.

    • Κάτω αριστερά—Τοποθέτηση συρραφής στην κάτω αριστερή γωνία των σελίδων.

    • Κάτω δεξιά—Τοποθέτηση συρραφής στην κάτω δεξιά γωνία των σελίδων.

    • Διπλή επάνω—Τοποθέτηση δύο συρραφών στην επάνω άκρη των σελίδων.

    • Διπλή αριστερά—Τοποθέτηση δύο συρραφών στην αριστερή άκρη των σελίδων.

    • Διπλή κάτω—Τοποθέτηση δύο συρραφών στην κάτω άκρη των σελίδων.

    • Διπλή δεξιά—Τοποθέτηση δύο συρραφών στη δεξιά άκρη των σελίδων.

    Οι παρακάτω επιλογές είναι διαθέσιμες για τη μονάδα συρραφής και τη μονάδα διάτρησης:

    Σημείωση: Η συρραφή τοποθετείται στην αριστερή άκρη των σελίδων.

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Απενεργοποίηση—Απενεργοποιεί τη λειτουργία συρραφής.

    • Αυτόματα—Αυτόματη τοποθέτηση συρραφής.

    • Μπροστά—Τοποθέτηση συρραφής μπροστά.

    • Πίσω—Τοποθέτηση συρραφής πίσω.

    • Διπλή—Τοποθέτηση δύο συρραφών.
