컬러 보정 설정

이 기능을 사용하면 다른 출력 장치 또는 표준 컬러 시스템의 색상과 일치하도록 인쇄 색상을 조정할 수 있습니다.

참고: 이 기능은 컬러 프린터에서만 사용할 수 있습니다.

  1. 옵션 메뉴에서 컬러를 선택한 후 컬러 보정을 클릭합니다.

  2. 컬러 보정 메뉴에서 다음 중 하나를 선택합니다.

    • 프린터 설정 사용—프린터에 지정된 설정을 사용합니다.

    • 끄기 - 색상 보정 기능을 해제합니다.

    • 자동—인쇄 페이지의 각 개체에 다른 색상 변환 테이블을 적용합니다.

    • 수동—인쇄된 페이지의 각 개체에 적용되는 색상 변환표를 사용자 정의하고 RGB 및 CMYK 컬러 보정 설정을 활성화합니다.

    참고: 사용되는 색상 변환 테이블은 개체 유형 및 각 개체의 색상이 지정되는 방식에 따라 달라집니다.

  3. 수동을 선택한 경우 각 RGB 및 CMYK 컬러 보정 설정에 대한 옵션을 선택합니다.

    다음 옵션을 이미지, 텍스트 및 그래픽 RGB 컬러 보정에 사용할 수 있습니다.

    • 프린터 설정 사용—프린터에 지정된 설정을 사용합니다.

    • 꺼짐—색상 변환을 적용하지 않습니다.

    • sRGB 디스플레이—이미지를 컴퓨터 모니터에 표시되는 방식으로 인쇄합니다. 그래픽을 인쇄하는 경우 이 설정을 권장합니다.

    • sRGB Vivid—이미지를 컴퓨터 모니터에 표시되는 것 보다 채도를 높게 인쇄합니다. 비즈니스 그래픽 및 텍스트를 인쇄할 경우 이 설정을 권장합니다.

    • 디스플레이 ‑ 순수 검정—검정색 토너만을 사용하여 중간 회색을 인쇄합니다.

    • Vivid—이미지의 밝기와 채도를 높게 인쇄합니다.

    다음 옵션은 CMYK 컬러 보정에 사용할 수 있습니다.

    • 프린터 설정 사용—프린터에 지정된 설정을 사용합니다.

    • 꺼짐—색상 변환을 적용하지 않습니다.

    • Vivid CMYK—이미지를 SWOP(Specifications for Web Offset Publishing) 색상 변환 테이블을 사용하여 인쇄합니다.

    • Euro CMYK—이미지를 EuroScale 색상 변환 테이블을 사용하여 인쇄합니다.

    • US CMYK—U.S. CMYK 색상 변환 테이블을 사용할 경우 이미지의 채도가 더 높게 인쇄됩니다.

  4. 인쇄를 클릭합니다.
