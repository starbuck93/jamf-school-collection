Auswählen einer Papierablage

Hinweis: Bevor Sie andere als die zum Lieferumfang des Druckers gehörenden Ablagen verwenden, müssen Sie sicherstellen, dass Ihr Drucker dies unterstützt. Weitere Informationen finden Sie im Benutzerhandbuch des Druckers.

  1. Wählen Sie im Options-Menü die Option Papierausgabe.

  2. Legen Sie unter "Papierablage" fest, welche Ablage verwendet werden soll.

    • Druckereinstellung verwenden: Es wird die im Drucker festgelegte Einstellung verwendet.

    • Standardschacht: Es wird die Standardablage des Druckers verwendet.

    • Schacht [x]: Es wird eine optionale Ablage des Druckers verwendet.

  3. Klicken Sie auf Drucken.
