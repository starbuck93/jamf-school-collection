Configuración de las correcciones de color

Esta función ajusta los colores impresos para que se correspondan con los colores de otros dispositivos de salida o sistemas de color estándar.

Nota: Esta función está disponible únicamente en impresoras en color.

  1. En el menú Opciones, seleccione Color y, a continuación, haga clic en Correción de color.

  2. En el menú Corrección de color, seleccione una de las siguientes opciones:

    • Usar valores de la impresora: Utilizar el valor especificado en la impresora.

    • Desactivar: Deshabilitar la corrección de color.

    • Automático: Aplicar una tabla de conversión de color diferente a cada objeto de la página impresa.

    • Manual: Personalizar la tabla de conversión de color que se aplica a cada objeto en la página impresa y activa los valores de la corrección de color configuración de RGB y CMYK.

    Nota: La tabla de conversión de color utilizada depende del tipo de objeto y la forma en la que se especifica el color para cada objeto.

  3. Si se selecciona Manual, seleccione una opción para cada valor de la corrección de color RGB y CMYK.

    Están disponibles las siguientes opciones para las correcciones de color de RGB para imágenes, texto y gráficos:

    • Usar valores de la impresora: Utilizar el valor especificado en la impresora.

    • Desactivado: No se aplica la corrección de color.

    • Pantalla sRGB:Imprimir las imágenes para que tengan un aspecto muy similar al del monitor de un ordenador. Se recomienda seleccionar este valor al imprimir gráficos.

    • sRGB intenso: Imprimir las imágenes para que estén más saturadas que las que aparecen en el monitor de un ordenador. Se recomienda este valor al imprimir texto y gráficos de negocio.

    • Pantalla ‑ Negro verdadero: Imprimir colores grises neutros utilizando solo tóner negro.

    • Intenso: Imprimir imágenes más brillantes y más saturadas.

    Están disponibles las siguientes opciones para la corrección de color CMYK:

    • Usar valores de la impresora: Utilizar el valor especificado en la impresora.

    • Desactivado: No se aplica la corrección de color.

    • CMYK intenso: Imprimir imágenes según las especificaciones de la tabla de conversión de color SWOP (del inglés, Specifications for Web Offset Publishing, especificaciones para publicaciones Web).

    • CMYK EURO: Imprimir imágenes según la tabla de conversión de color Euroscale.

    • CMYK EE. UU.: Imprimir imágenes más saturadas al utilizar la tabla de conversión de color CMYK EE. UU.

  4. Haga clic en Imprimir.
