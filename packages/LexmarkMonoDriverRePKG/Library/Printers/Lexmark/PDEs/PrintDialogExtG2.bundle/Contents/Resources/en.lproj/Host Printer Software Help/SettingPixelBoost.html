Setting pixel boost

This feature enhances the image and text detail of the printed document by slightly increasing pixel density according to the selected setting.

Note: This option is available only in monochrome printers.

  1. In the options menu, select Quality.

  2. In the Pixel Boost menu, select one of the following:

    • Use Printer Setting—Use the setting that is specified in the printer.

    • Off—Disable pixel boost.

    • Fonts—Darken the text on the printed document. This setting affects only font data.

    • Horizontally—Darken the horizontal lines of image and text on the printed document.

    • Vertically—Darken the vertical lines of image and text on the printed document.

    • Both Directions—Darken both horizontal and vertical lines of image and text on the printed document.

  3. Click Print.
