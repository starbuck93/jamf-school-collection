Использование функции "Учет заданий"

Данная функция позволяет отслеживать задания печати, связывая их с именем пользователя или идентификатором, кодом учета и кодом отдела. Благодаря связи заданий печати с определенными данными учета имеется также возможность установить квоты по заданиям печати, которые могут отправлять пользователи. Чтобы установить квоты по пользователям, установите на принтере "Квоты устройства". Подробнее см. в Руководстве администратора по квотам устройства.

  1. В меню параметров выберите Учет заданий >Включить учет заданий.

  2. Введите или выберите имя пользователя или идентификатор, код учета и код отдела.

    Примечание. По умолчанию используется имя пользователя, под которым выполнен вход в систему. Для создания пользовательского имени очистите поле Использовать имя пользователя, вошедшего в систему, затем введите имя пользователя.

  3. Нажмите Печать.
