設定色彩平衡

此功能功能可讓您分別增加或減少每個色彩平面所使用的碳粉量，來調整列印輸出的色彩。

請注意： 此功能只適用於彩色印表機。

  1. 在選項功能表中，選取彩色，然後按一下色彩平衡。

  2. 在「色彩平衡」功能表中，選取下列其中一項：

    • 使用印表機設定—使用印表機中指定的設定值。

    • 手動—自訂列印輸出中所使用的靛青色、洋紅色、鮮黃色和黑色碳粉量。

  3. 若是選取了手動，請使用滑桿調整下列設定：

    • 紅色—調整紅色與靛青色之間的色彩平衡。

    • 綠色—調整綠色與洋紅色之間的色彩平衡。

    • 藍色—調整藍色與鮮黃色之間的色彩平衡。

    • 白色—調整白色與黑色之間的色彩平衡。

  4. 按一下列印。
