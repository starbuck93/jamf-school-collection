Skriva ut häften

Med den här funktionen kan du skriva ut ett flersidigt dokument som kan vikas till ett häfte.

  1. Välj Avancerad layout på alternativmenyn.

  2. Välj Häfte på menyn Layout.

  3. På meny Ark per bunt anger du antalet sidor som finns i varje vikt bunt.

    Obs! Genom att ställa in det här alternativet på ett lämpligt antal blir bunten lättare att vika och papperets kanter ligger jämnt.

  4. Om det behövs väljer du Bibehåll originalstorleken på häftets sidor för att ändra storleken på det utskrivna häftet.

    Anmärkningar:
    • Om du väljer Bibehåll originalstorleken på häftets sidor får varje sida i häftet samma storlek som originaldokumentet. Om originaldokumentet exempelvis är 210 x 297 mm skrivs häftet ut på papper med måtten 279 x 432 mm. Genom att skriva ut på ett papper med måtten 279 x 432 mm bibehålls måtten på originaldokumentet. Inställningen fungerar bäst på originaldokument med pappersstorlekarna statement, letter, A5, A4 och B5.
    • Om du inte väljer Bibehåll originalstorleken på häftets sidor skalas de enskilda sidorna i häftet ned till originaldokumentets pappersstorlek.

  5. Om det behövs väljer du Inkludera häftesomslag för att aktivera tillägg av omslag och väljer sedan ett eller flera av följande alternativ:

    • Skriv ut på framsidans utsida

    • Skriv ut på framsidans insida

    • Skriv ut på baksidans utsida

    • Skriv ut på baksidans insida

  6. Välj papperstyp för häftesomslag på menyn Papperstyp för omslag.

  7. Välj vilket fack som ska användas för häftesomslaget på menyn Papperskälla för omslag.

  8. Klicka på Skriv ut.
