Selecting an output bin

Note: To use bins other than the bins that originally came with your printer, make sure that your printer supports optional bins. For more information, see the printer User’s Guide.

  1. In the options menu, select Finishing.

  2. In the Output Bin menu, select the bin that you want to use.

    • Use Printer Setting—Use the setting that is specified in the printer.

    • Standard Bin—Use the standard bin of the printer.

    • Bin [x]—Use an optional bin that is available for your printer.

  3. Click Print.
