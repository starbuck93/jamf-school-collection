Настройка плотности тонера

Функция позволяет осветлить или затемнить распечатки.

  1. В меню параметров выберите Качество.

  2. В меню "Плотность тонера" выберите одно из следующего:

    • Использовать настройки принтера — Использование настроек, заданных на принтере.

    • Вручную — настройка плотности тонера печатаемого документа.

  3. Ползунками выбирается значение плотности тонера.

  4. Нажмите Печать.
