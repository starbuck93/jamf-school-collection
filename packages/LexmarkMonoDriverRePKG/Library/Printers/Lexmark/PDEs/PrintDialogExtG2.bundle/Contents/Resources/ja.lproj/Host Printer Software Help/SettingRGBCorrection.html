RGB 補正を設定する

この機能を使用すると、印刷出力の明るさ、コントラスト、彩度を調整できます。

メモ： この機能は、カラープリンタでのみ使用できます。

  1. ［オプション］メニューで［カラー］を選択し、［RGB 補正］をクリックします。

  2. ［RGB 補正］メニューより、以下の中からいずれか 1 つを選択します。

    • ［プリンタの設定を使用］ - プリンタで指定された設定を使用します。

    • ［手動］ - 印刷出力の明るさ、コントラスト、および彩度をカスタマイズします。

  3. ［手動］を選択した場合、スライドバーを使用して次の設定を調整します。

    • ［明るさ］ - 印刷出力の明るさを調整します。

    • ［コントラスト］ - 印刷出力のコントラストを調整します。

    • ［彩度］ - 印刷出力の鮮やかさとカラー濃度を調整します。

  4. ［印刷］をクリックします。
