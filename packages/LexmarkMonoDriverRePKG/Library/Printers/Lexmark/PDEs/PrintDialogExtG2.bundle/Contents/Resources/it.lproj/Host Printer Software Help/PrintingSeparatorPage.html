Stampa di una pagina di separazione

Utilizzare questa funzione per separare un documento da un altro durante la stampa di più copie di un documento o più documenti in un unico processo di stampa.

  1. Nel menu Opzioni, selezionare Finitura.

  2. Nel menu Origine separatore, selezionare il vassoio della pagina di separazione.

  3. Nel menu Pagina di separazione, selezionare quando stampare la pagina di separazione.

    • Usa impostazione stampante: consente di utilizzare le impostazioni specificate nella stampante.

    • Nessuna: non viene stampata nessuna pagina di separazione.

    • Tra processi: consente di stampare una pagina di separazione al termine della stampa di un processo.

    • Tra copie: consente di stampare una pagina di separazione al termine della stampa di una copia del processo di stampa.

  4. Fare clic su Stampa.
