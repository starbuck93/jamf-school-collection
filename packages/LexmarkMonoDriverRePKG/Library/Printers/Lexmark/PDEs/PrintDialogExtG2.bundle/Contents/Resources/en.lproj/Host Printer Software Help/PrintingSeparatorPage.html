Printing a separator page

Use this feature to separate one document from another when printing multiple documents or copies of the document in a single print job.

  1. In the options menu, select Finishing.

  2. In the Separator Source menu, select the tray of the separator page.

  3. In the Separator Page menu, select when to print a separator page.

    • Use Printer Setting—Use the setting that is specified in the printer.

    • None—Do not print a separator page.

    • Between Jobs—Print a separator page after a print job has finished printing.

    • Between Copies—Print a separator page after a copy of the print job has finished printing.

  4. Click Print.
