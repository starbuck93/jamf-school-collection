Falten von Ausdrucken

Hinweis: Vergewissern Sie sich, dass Ihr Drucker über einen Falt-Finisher verfügt. Weitere Informationen finden Sie im Benutzerhandbuch des Druckers.

  1. Wählen Sie im Options-Menü die Option Papierausgabe.

  2. Legen Sie im Menü "Falten" fest, wie das Dokument gefaltet werden soll.

    • Aus: Deaktiviert die Faltfunktion.

    • Falten: Faltet bis zu fünf Blätter automatisch in der Mitte, kurze Kante auf kurze Kante, so dass ein Bündel entsteht. Bei Druckaufträgen mit mehr als fünf Blättern wird jedes 5-Blatt-Bündel in das nächste Bündel eingelegt.

    • Falten und in der Mitte heften: Faltet das Dokument automatisch in der Mitte, kurze Kante auf kurze Kante, und heftet die Blätter entlang der Falz.

      Hinweis: Vergewissern Sie sich, dass Ihr Drucker über einen Hefter-Finisher verfügt. Weitere Informationen finden Sie im Benutzerhandbuch des Druckers.

    • Einzelne Seiten falten: Faltet einzelne Seiten automatisch in der Mitte, kurze Kante auf kurze Kante.

    • Einzelne Seiten dreifach falten: Faltet einzelne Seiten automatisch dreifach, kurze Kante auf kurze Kante.

    Hinweis: Diese Optionen sind nur auf einigen Druckermodellen verfügbar.

  3. Klicken Sie auf Drucken.
