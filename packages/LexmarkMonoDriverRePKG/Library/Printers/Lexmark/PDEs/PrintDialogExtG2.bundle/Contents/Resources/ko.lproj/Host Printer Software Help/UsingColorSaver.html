색상 절약기 사용

이 기능은 컬러 이미지를 인쇄할 때 프린터에서 사용하는 토너량을 줄입니다.

참고: 이 기능은 컬러 프린터에서만 사용할 수 있습니다.

  1. 옵션 메뉴에서 컬러를 선택한 후 컬러를 클릭합니다.

  2. ColorSaverTM 메뉴에서 다음 중 하나를 선택합니다.

    • 프린터 설정 사용—프린터에 지정된 설정을 사용합니다.

    • 켜짐—ColorSaver을(를) 활성화합니다.

    • 꺼짐—ColorSaver을(를) 비활성화합니다.

    참고: 텍스트는 기본량의 토너를 사용하여 인쇄됩니다.

  3. 인쇄를 클릭합니다.
