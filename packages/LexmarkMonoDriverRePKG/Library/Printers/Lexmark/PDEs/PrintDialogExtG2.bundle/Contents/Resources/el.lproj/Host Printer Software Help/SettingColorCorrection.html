Ρύθμιση διορθώσεων χρώματος

Αυτή η λειτουργία ρυθμίζει τα εκτυπωμένα χρώματα για να ταιριάζουν καλύτερα με τα χρώματα άλλων συσκευών εξόδου ή τυπικών συστημάτων χρωμάτων.

Σημείωση: Αυτή η λειτουργία διατίθεται μόνο σε έγχρωμους εκτυπωτές.

  1. Στο μενού Επιλογές, επιλέξτε Χρώμα και, στη συνέχεια, κάντε κλικ στην επιλογή Διόρθωση χρώματος.

  2. Στο μενού Διόρθωση χρώματος, επιλέξτε ένα από τα παρακάτω:

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Απενεργοποίηση—Απενεργοποίηση της διόρθωσης χρώματος.

    • Αυτόματα—Εφαρμόζει διάφορους πίνακες μετατροπής χρώματος σε κάθε αντικείμενο της εκτυπωμένης σελίδας.

    • Μη αυτόματα—Προσαρμόζει τον πίνακα μετατροπής χρώματος που εφαρμόζεται σε κάθε αντικείμενο της εκτυπωμένης σελίδας και ενεργοποιεί τις ρυθμίσεις διόρθωσης χρώματος RGB και CMYK.

    Σημείωση: Ο πίνακας μετατροπής χρώματος που χρησιμοποιείται εξαρτάται από τον τύπο αντικειμένου και τον τρόπο καθορισμού του χρώματος για κάθε αντικείμενο.

  3. Εάν έχετε επιλέξει Μη αυτόματα, ορίστε μια επιλογή για κάθε ρύθμιση διόρθωσης χρώματος RGB και CMYK.

    Οι παρακάτω επιλογές διατίθενται για διορθώσεις χρώματος RGB γραφικών, εικόνων και κειμένου:

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Απενεργοποίηση—Να μην εφαρμοστεί μετατροπή χρωμάτων.

    • Οθόνη sRGB—Εκτύπωση εικόνων ώστε να φαίνεται ο τρόπος με τον οποίο θα εμφανίζονται στην οθόνη του υπολογιστή. Αυτή η ρύθμιση συνιστάται κατά την εκτύπωση γραφικών.

    • Ζωντανό sRGB—Εκτύπωση εικόνων με μεγαλύτερο κορεσμό από αυτόν που εμφανίζεται στην οθόνη του υπολογιστή. Αυτή η ρύθμιση συνιστάται κατά την εκτύπωση επαγγελματικών γραφικών και κειμένου.

    • Οθόνη - Φυσικό μαύρο—Εκτύπωση χρωμάτων ουδέτερου γκρι, με τη χρήση μόνο του μαύρου τόνερ.

    • Ζωντανό—Εκτύπωση εικόνων ώστε να είναι πιο φωτεινές και κορεσμένες.

    Οι παρακάτω επιλογές διατίθενται για τη διόρθωση χρώματος CMYK:

    • Χρήση ρύθμισης εκτυπωτή—Χρησιμοποιεί τη ρύθμιση που καθορίζεται στον εκτυπωτή.

    • Απενεργοποίηση—Να μην εφαρμοστεί μετατροπή χρωμάτων.

    • Ζωντανό CMYK—Εκτύπωση εικόνων με τις προδιαγραφές του πίνακα μετατροπής χρώματος για εκτυπώσεις web offset (SWOP).

    • Euro CMYK—Εκτύπωση εικόνων με τη χρήση πίνακα μετατροπής χρώματος Euroscale.

    • US CMYK—Εκτύπωση πιο κορεσμένων εικόνων κατά τη χρήση του πίνακα μετατροπής χρώματος US CMYK.

  4. Κάντε κλικ στην επιλογή Εκτύπωση.
